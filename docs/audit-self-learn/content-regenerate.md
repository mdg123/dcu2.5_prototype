# 자동생성 quiz 콘텐츠 재생성 리포트

작성일: 2026-04-28
대상: `contents.tags LIKE '%자동생성%'` 인 quiz 콘텐츠 1,348개
스크립트: `scripts/regenerate-quiz-content.js` (v2 generator)

---

## 1. 배경

기존 `scripts/fill-missing-questions.js` 의 `generateProblem()` 함수는 lesson_name 키워드 매칭이 단순(정규식 분기 ~14개)하고, **`learning_map_nodes.grade_level('초/중/고')`을 사용하지 않고 `grade` 컬럼만 사용**하여 다음과 같은 치명적 오류를 발생시켰다.

| 노드 | 기대 | 기존 출제 |
|-----|------|----------|
| 고1 "여집합" | 집합 연산 | `1+6=?` |
| 고1 "여집합과 차집합의 성질" | 집합 연산 | `1+6=?` |
| 중1 "직선, 반직선, 선분의 기호" | 도형 정의 | `변이 3개인 도형의 이름은?` |
| 중2 "일차함수의 뜻" | 함수 그래프/기울기 | 초등 수 배열 패턴 |
| 초3 "정육면체의 면, 모서리, 꼭짓점의 개수" | 6/12/8 | `변이 5개 도형은?` |

원인: `grade_level='고'` 인 노드도 `grade=1` 이라 elementary fallback `genCounting/genAddition` 로 폴백되었음.

감리/교사테스터 보고서에 따르면 자동생성 1,348건 중 **약 65%가 노드와 무관**.

## 2. 변경 사항

### 2.1 신규 스크립트
- `scripts/regenerate-quiz-content.js` 추가

  - 옵션:
    - `--dry` : 핸들러 분포 통계만 출력

    - 기본 실행 시 1,348개 전부 재생성

### 2.2 generateProblemV2 의 분기 구조

```
if 중·고:
  → 여집합/차집합/교집합/합집합/드모르간/벤다이어그램  →  genSetOp
  → 명제의 부정 / 충분조건·필요조건                  →  genProposition
  → 일차함수 / 이차함수                              →  genLinearFn / genQuadraticFn
  → 합성함수 / 역함수                                →  genFunctionConcept
  → 선분/반직선/직선 (기호·뜻·정의)                  →  genLineKind
  → 정수와 유리수의 (덧/뺄/곱/나)                     →  genIntegerOp (음수 보장)
  → 소인수분해 / 거듭제곱                            →  genFactorization
  → 두 점 사이의 거리                                →  genDistance (피타고라스 정수해)
  → 순열·조합·계승·합/곱의 법칙                      →  genPermComb
  → 그 외                                            →  genConceptFallback ← 임의 산술 X

if 초:
  → 시각/시계/몇 시 몇 분                            →  시각 읽기 MCQ
  → kg, g, 무게                                      →  무게 비교 MCQ
  → 들이/L/mL                                        →  들이 합 MCQ
  → 넓이/cm²/m²                                      →  직사각형 넓이 MCQ
  → 길이/cm/mm/km                                    →  genMeasurement (길이 비교)
  → 각도/예각/둔각/직각                              →  각의 종류 분류
  → 정육면체/직육면체                                →  genCube (면 6 / 모서리 12 / 꼭짓점 8)
  → 분수 + 덧셈/뺄셈/곱셈                            →  분수 사칙연산 MCQ
  → 분수/약분/통분/분모/분자                         →  genFraction
  → 자릿값/큰 수/억/조                               →  genPlaceValue
  → 곱셈|×, 나눗셈|÷|몫                              →  genMultiplication / genDivision
  → 0이 있는 덧셈/뺄셈                               →  +0 / -0 MCQ
  → 덧셈|첨가|합병, 뺄셈|제거                        →  genAddition / genSubtraction
  → 세기, 읽고쓰기, 크기 비교, 9까지/50까지/100까지  →  각 전용 MCQ
  → 쌓기나무                                          →  concept-fallback (임의 도형 출제 방지)
  → 원/원인 것/원이 아닌/지름/반지름/원주             →  원 정의 MCQ
  → 삼각형~육각형/다각형/평면도형                    →  genShapeBySides
  → 원기둥/원뿔/구/입체도형                          →  입체도형 식별
  → 각기둥/각뿔 (면/모서리/꼭짓점)                   →  공식 기반 MCQ (n각기둥 = 면 n+2/모서리 3n/꼭짓점 2n)
  → 도형 일반                                         →  genShapeBySides
  → 규칙/배열/대응/수 배열                            →  genPattern
  → 가능성 + (말/표현/확실/불가능)                   →  가능성 5단계 MCQ
  → 평균                                              →  평균 계산
  → 그래프/자료/분류/통계/확률                       →  genGraph
  → area 폴백 (도형/변화/자료)                       →  영역 기본 핸들러
  → 그 외                                            →  genConceptFallback
```

### 2.3 폴백 정책 변경
- **기존**: 매칭 실패 시 `grade <= 1 → 수 세기`, `grade <= 2 → 덧셈`, … 로 무관한 산술 강제 출제
- **신규**: `genConceptFallback(lname, area)` — 같은 area 의 다른 lesson_name 을 distractor 로 한 **개념 식별 MCQ**. 정답이 lesson_name 자체이므로 학생 입장에서 항상 본 차시 주제를 인지 가능.

## 5. 패턴 매칭 추가 항목 (요약)

신규 분기 (v1 → v2):
- 학교급(`grade_level`) 우선 디스패치
- 집합 연산 (여집합·차집합·교집합·합집합·드 모르간)
- 명제 (부정·충분/필요/필요충분 조건)
- 일차/이차함수 (기울기·y절편·함숫값 / x²)
- 합성함수 / 역함수
- 정수와 유리수의 사칙연산 (음수 보장)
- 소인수분해 / 거듭제곱 표현
- 좌표평면 두 점 사이의 거리 (피타고라스 정수해)
- 순열·조합·계승·합/곱의 법칙
- 선분/직선/반직선 정의
- 정육면체/직육면체 (면 6 / 모서리 12 / 꼭짓점 8)
- 각기둥/각뿔 (n에 따른 면·모서리·꼭짓점 공식)
- 시각 읽기 (몇 시 몇 분)
- 무게 비교 (kg/g)
- 들이 합 (L)
- 직사각형 넓이 (cm²)
- 각의 종류 (예각/직각/둔각)
- 분수 덧셈·뺄셈·(분수)×(자연수)
- 0이 있는 덧셈·뺄셈
- 가능성 5단계 (확실/일 것 같다/반반/아닐 것 같다/불가능)
- 평균 (4개 수의 평균)
- 원의 정의 (중심으로부터 같은 거리)
- 쌓기나무 (개념 폴백 — 임의 도형 출제 차단)
- **개념형 폴백** (모든 미매칭 케이스 — 무관한 산술 출제 차단)

## 6. 후속 권장 사항

1. 중·고등 핵심 690건이 여전히 개념-폴백에 의존. 영역별 추가 핸들러(예: 다항식 사칙연산, 복소수, 도형의 방정식, 행렬 사칙, 유리/무리함수 그래프) 단계적 보강 필요.

## 7. 실행 명령

```bash
# 통계만 (DB 변경 없음)
node scripts/regenerate-quiz-content.js --dry

# 무작위 30개 샘플 출력
node scripts/regenerate-quiz-content.js --sample

# 본 실행 (1,348개 일괄 갱신)
node scripts/regenerate-quiz-content.js
```

스크립트는 멱등(idempotent)이며, content_id는 동일하게 유지되므로 외부 참조(node_contents, content_content_nodes 등) 무결성을 해치지 않는다.

## 1. 배경 — v2의 P1 결함

v2 재생성에서 폴백 핸들러(`genConceptFallback`)가 **영역별 단일 템플릿** 사용:

```
"다음 중 OO 영역에서 본 차시의 학습 주제로 가장 알맞은 것은?"
```

690건에 동일 패턴 적용 → 자동생성 1,348건의 **distinct question_text 1,187 → 1,049 로 감소**(중복도 악화). 단일 문항 최대 **182회 재사용** (`도형과 측정` 영역).

DB 실측 (v2 종료 직후):
```
SELECT COUNT(DISTINCT question_text) FROM content_questions q
JOIN contents c ON c.id=q.content_id WHERE c.tags LIKE '%자동생성%';
→ distinct: 445 / total: 1348  (33.0%)
```

v2 dry-run 통계의 "concept-fallback 690" 은 sibling 풀 다양도가 매우 낮은(영역당 unique question_text 1개) 케이스가 많아 실제 DB distinct 는 445로 더 떨어짐.

## 2. 변경 사항 (v2 → v3)

### 2.1 신규 스크립트 — `scripts/regenerate-quiz-v3.js`

- v2 generator 의 `generateProblemV2` 를 그대로 호출 (정상 핸들러 612건은 보존)
- 결과 텍스트가 폴백 패턴(`/학습 주제로 가장 알맞은/`) 인 경우에만 **v3 다양화 라우터** 적용
- 노드별 결정성 시드 `tpl_${node_id}_${content_id}` 로 6종 템플릿 중 1종 선택 → 재실행 시 동일 결과

### 2.2 6종 폴백 템플릿

| 코드 | 이름 | 설명 |
|------|------|------|
| **T1** | 정의형 | "[OO 영역] 다음 중 본 차시의 학습 주제로 가장 알맞은 것은?" — 같은 영역의 다른 lesson_name distractor (v2 호환 문구 유지하되 영역 prefix 만 보유) |
| **T2** | 단원 구별형 | "「단원명」 단원의 학습 주제 중 본 차시(영역)에 해당하는 것은?" — 같은 `unit_name` 의 다른 lesson 으로 distractor (메타정보 활용) |
| **T3** | 빈칸 채우기 | lname 에 핵심어가 포함된 경우 그 부분을 `___` 로 마스킹 / 미포함 시 "OO 영역의 핵심 개념은 ___이다" 형식 |
| **T4** | 예시·사례형 | lname 과 매칭되는 핵심어의 실생활 예시(KEYWORDS_BY_AREA_LEVEL.examples) 를 정답으로, 다른 핵심어의 예시를 distractor 로 |
| **T5** | 메타인지 매칭형 | "lname 은 어느 학년·영역에서 다루는 주제입니까?" — 학년×영역 라벨 4지선다 |
| **T6** | 풀이형 | "OO(핵심어)을 가장 잘 설명한 것은?" — 영역별 정의를 정답으로, 타 영역 정의를 distractor 로 |

## 5. 데이터 무결성 검증

- options 비어있음 / 손상: 0
- options 길이 < 3: 0
- answer 인덱스 범위 초과: 0
- 6종 템플릿 모두 `Array.isArray(options) && options.length >= 4` 가드 통과

## 6. 후속 권장 사항

1. **T1 잔존 중복 35건(도형과 측정)**: T1 의 sibling 풀에 의존하므로 영역의 unique 다양성 한계. 영역 내 유사 lesson 그룹화로 distractor 풀 다변화 필요. (P2)
2. **v2 약매칭 7건**: `소수 사이의 관계`, `(대분수)÷(분수)`, `(자연수)÷(소수)` 같이 키워드만 매칭되어 산술로 빠지는 케이스 — 별도 핸들러 추가 또는 v2 매칭 정밀화 (P1).
3. **T4 distractor 결손**: `KEYWORDS_BY_AREA_LEVEL.examples` 의 keyword 별 예시 풀이 평균 1~2개 — 4지 distractor 채우기 위해 영역당 ≥ 3 keyword × ≥ 2 example 보강 권고 (P2).
4. **EBS/외부 콘텐츠 연계**: 폴백 736건 중 핵심 차시는 EBS·자체제작 학습영상으로 우선 대체하여 폴백 비중을 < 30% 로 낮추는 중장기 전략 (P2).

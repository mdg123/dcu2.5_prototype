# 다채움 LRS 대시보드 개선 제안서
## 교육부 AIDT 학습분석 지원 서비스 벤치마킹 기반

---

## 1. 벤치마킹 대상 개요

교육부 AI·디지털 교육자료 학습분석 지원 서비스(LAS)는 전국 시도교육청 단위로 학생들의 AI·디지털 교육자료 활용 현황을 종합적으로 분석·제공하는 대시보드 서비스이다. 
해당 서비스는 학습현황, 학업성취, 학습수행, 맞춤학습, 교사활용지수, 일일현황의 6개 대분류 아래 약 40여 개의 세부 조회 화면을 갖추고 있으며, 
주차·월간·학기별·연간 등 다양한 기간 단위와 교육청·학교급·과목별 다차원 필터링 기능을 제공한다. 다채움 LRS 대시보드가 이와 유사한 수준의 학습분석 정보를 제공할 수 있도록, 아래와 같이 기능별 개선 사항을 제안한다.

---

## 2. 학습현황 분석 대시보드

### □ 활용빈도 및 접속률 분석

○ 전체 활용률 추이 시각화

• 주차별 활용률(%) 꺾은선 그래프를 통해 전체 학습자의 LRS 콘텐츠 접속률 추이를 한눈에 파악할 수 있도록 대시보드 상단에 배치

• 활용률 산출식(접속건수 총합 / 약관동의 학생수 × 100)을 화면 내 산출식 영역에 명시하여 데이터 해석의 투명성 확보

• 표 영역에서 순번, 기간(주차), 소속기관명, 활용률(%)을 컬럼으로 제공하고, 엑셀 다운로드 기능을 통해 상세 데이터를 외부 분석에 활용 가능하도록 지원

○ 활용빈도 합계·평균 분석

• 주차별 접속 로그 기반의 활용빈도(건수)를 합계 또는 평균 방식으로 전환하여 조회할 수 있는 계산방식 토글 제공

• 합계 방식은 전체 접속 규모를 파악하는 데 활용하고, 평균 방식은 학생 1인당 접속빈도를 파악하여 학습 참여 밀도를 분석하는 데 활용

• 과목별 활용빈도를 다중 라인 그래프로 시각화하여 과목 간 활용 격차를 직관적으로 비교

• 학교급별(초등학교, 중학교, 고등학교) 활용빈도를 별도 그래프 및 표로 제공하여 학교급에 따른 활용 패턴 차이를 분석 가능하도록 구성

○ 주간 과목시수 반영 옵션

• 과목별 주간 배정 시수를 반영한 보정 활용빈도 산출 기능을 제공하여, 수업 시수가 많은 과목의 활용빈도가 과대평가되지 않도록 보정

• 학년별 집계 체크박스를 통해 학교급 내에서 학년별 상세 분석으로의 드릴다운 지원

### □ 활용시간 분석

○ 학습활동 시간 추이 시각화

• 주차별 학습활동 시간(h)을 꺾은선 그래프로 제공하여, 시간 경과에 따른 학습 투입 시간의 변화 추이를 파악

• 합계/평균 전환 기능을 통해 전체 학습시간 규모와 학생 1인당 평균 학습시간을 모두 분석 가능하도록 구성

• 과목별 학습활동 시간을 과목 라인별로 구분하여 시각화하고, 표에서 과목별 시간 데이터를 상세 제공

• 학교급별 학습활동 시간을 초등학교, 중학교, 고등학교별로 분리 시각화하여, 학교급에 따른 학습 시간 차이를 정량적으로 분석

○ 기간 유연성 확보

• 일간, 월간, 학기별, 연간 등 다양한 기간 단위 조회를 지원하는 라디오 버튼 그룹 제공

• 시작일과 종료일을 달력으로 직접 선택할 수 있는 커스텀 기간 조회 기능 추가

• 학년도 드롭다운을 통해 연도별 비교 분석이 가능하도록 지원

## 3. 학업성취 분석 대시보드

### □ 성취수준 현황 및 분포

○ 교육과정 표준체계 기반 성취수준 현황

• 교육과정 표준체계 ID, 내용체계 영역, 단계별 내용요소를 행(Row) 구조로 배치하여, 성취기준 단위의 학습 달성 현황을 표로 제공

• 성취수준을 A, B, C 등급으로 분류하여 각 성취기준별 학생 분포를 수치로 표시하고, 합계 행을 통해 전체 달성 현황을 요약

• 과목과 학교급(초등학교, 중학교, 고등학교)을 필수 필터로 설정하여 과목·학교급 조합별 성취 현황 조회 가능

• 기준일자 기간 설정에 1일, 1주일, 1개월, 6개월, 1년 바로가기 버튼을 제공하여 빠른 기간 전환 지원

○ 성취수준 분포 시각화

• 집속일자별 과목별 성취수준 비율(%)을 A, B, C, D, E 등급으로 구분하여 그래프와 표로 동시에 제공

• 성취수준 누적 건수를 A~E 등급별로 상세히 표시하여, 특정 기간 내 등급별 학생 수 변화를 추적 가능하도록 구성

• 출력유형(전체, 그래프, 표) 선택 기능을 통해 사용자가 원하는 형태로 데이터를 조회할 수 있도록 지원

---

## 4. 학습수행 분석 대시보드

### □ 학습콘텐츠 활용도

○ 콘텐츠 유형별 활용빈도·활용시간 분석

• 영상, 음원, 텍스트, 실습, 이미지, 기타 등 콘텐츠 유형별 활용빈도(건수)를 누적 막대 그래프로 시각화하여 유형 간 활용 비중 비교

• 동일 콘텐츠 유형 분류 체계를 활용시간(h) 분석에도 적용하여, 유형별 학습 투입 시간을 대비 분석 가능하도록 구성

• 과목별로 데이터를 분리 표시하여, 과목마다 주로 활용하는 콘텐츠 유형의 차이를 파악

• 학교급별 주간 과목시수 반영 옵션을 통해 수업 시수 대비 콘텐츠 활용 밀도를 보정 산출 지원

### □ 문항풀이 활용도

○ 평가문항풀이 빈도·시간 분석

• 평가문항풀이 건수와 문항풀이 시간을 활용구분 라디오 버튼으로 전환하여 단일 화면에서 빈도와 시간 모두를 분석 가능하도록 구성

• 과목별, 학교급별 평가문항풀이 빈도·시간을 별도 화면으로 제공하여 세분화된 분석 지원

• 합계/평균 계산방식 토글을 제공하여 전체 규모와 학생 1인당 수치를 모두 파악 가능하도록 구성

### □ 과제 수행도

○ 과제 출제·제출 빈도 및 수행률 분석

• 과제 출제 빈도를 과목별 일자별로 시각화하고, 합계 컬럼을 통해 전체 출제 규모를 요약

• 과제 제출 빈도를 동일한 구조로 제공하여 출제 대비 제출 비율을 간접적으로 비교 분석 가능하도록 구성

• 과제 수행률을 별도 화면으로 제공하여 과제 완료 현황을 정량적으로 파악

• 그래프유형 전환(꺾은선/막대)과 집계단위 전환(일별/주별/월별) 기능을 통해 다양한 관점에서 데이터를 조회 가능하도록 지원

---

## 5. 맞춤학습 분석 대시보드

### □ 스스로 채움 내 오늘의 학습, 오답노트, AI 맞춤학습 활용 분석

○ 학생 주도 평가 참여빈도·재평가·오답 재도전 분석

• 학생 주도 평가 참여빈도를 추적하여 자기주도 학습 의지를 정량적으로 측정

• 학생 주도 재평가 참여빈도를 별도 지표로 관리하여 반복학습 의지를 파악

• 오답 재도전 참여율을 산출하여 틀린 문제에 대한 재학습 행동을 분석하고, 이를 통해 학습 지속성 및 학습 전략 수립 역량을 간접적으로 측정

### □ 교사 콘텐츠 재구성

○ 교육자료 재구성 빈도 분석

• 교사가 채움콘텐츠 내 탑재된 기존 교육자료를 담기 후 재구성(수정·편집·조합)한 빈도를 추적하여, 교사의 능동적 콘텐츠 활용 수준을 파악

• 기간별·과목별 재구성 빈도 추이를 제공하여 교사의 맞춤형 교육 활동 패턴을 분석

### □ 교사 피드백 빈도

○ 교사 피드백 활동 분석

• 교사가 학생에게 제공한 피드백 건수를 추적하여, 교사-학생 간 상호작용의 빈도를 정량적으로 파악

• 피드백 빈도의 시간 추이를 시각화하여 피드백 활동의 지속성 및 변화 패턴을 모니터링

---

## 6. 교사활용지수 대시보드

### □ 교사 관점 활용빈도 분석

○ 교사 활용률 및 활용빈도

• 교사의 채움클래스, 채움콘텐츠 접속률을 별도로 산출하여, 학생 활용률과 비교 분석 가능하도록 구성

• 교사 활용빈도를 과목별, 학교급별로 세분화하여 교사의 디지털 도구 활용 패턴을 다차원적으로 분석

• 교사 활용 데이터를 학생 활용 데이터와 동일한 시각화 형태(꺾은선 그래프, 데이터 표)로 제공하여 양 측면의 비교 분석 용이성 확보

---

## 7. 일일현황 모니터링 대시보드

### □ 운영 현황 종합 모니터링

○ 접속·활용 현황 일일 추적

• 시군, 학교급, 학교별로 세분화하여 현황을 일일 단위로 추적하여 실시간 모니터링

• 사용자 및 학교별 접속 현황을 일일 단위로 제공하여 실시간 활성 사용자 현황을 모니터링

---

## 8. 공통 UI/UX 개선 사항

### □ 조회 조건 및 필터 체계

○ 다차원 필터링 시스템 구축

• 학년도 드롭다운, 기간(일간/월간/학기별/연간) 라디오 버튼, 시작일-종료일 달력 선택을 조합하여 유연한 기간 조회 체계 구축

• 교육청, 교육지원청, 학교, 학교급, 과목 등 다계층 필터를 조합하여 원하는 수준의 데이터를 정밀하게 조회할 수 있도록 구성

• 출력유형(전체/그래프/표) 선택 기능을 제공하여 사용자가 원하는 시각화 형태를 선택 가능

• 조회조건 초기화 버튼을 제공하여 복잡한 필터 설정을 빠르게 리셋할 수 있도록 지원

### □ 데이터 시각화 및 내보내기

○ 그래프·표 연계 시각화 및 데이터 내보내기

• 모든 조회 화면에서 그래프와 표를 동시에 제공하여, 시각적 추세 파악과 정량적 상세 분석을 병행 가능하도록 구성

• 꺾은선 그래프, 막대 그래프, 누적 막대 그래프 등 데이터 특성에 맞는 차트 유형을 자동 적용하고, 필요 시 그래프유형 전환(꺾은선/막대) 지원

• 엑셀 다운로드 기능을 모든 표에 적용하여 외부 분석 도구에서의 2차 활용을 지원

• 화면설명 및 산출식 영역을 펼쳐보기/숨기기 방식으로 제공하여, 필요 시 데이터 산출 근거를 투명하게 확인 가능

### □ 대시보드 홈 화면

○ 핵심 KPI 요약 카드 및 요약 시각화

• 대시보드 홈 화면에 전체 활용빈도, 활용률, 전일대비 증감, 학교급별 요약 정보를 카드 형태로 배치하여 핵심 지표를 즉시 파악

• 공지사항 영역을 배치하여 시스템 관련 주요 안내를 즉시 확인 가능하도록 구성

---

## 9. 종합 요약

위 제안 내용을 종합하면, 다채움 LRS 대시보드는 교육부 AIDT 학습분석 지원 서비스의 구조를 벤치마킹하여 학습현황(활용빈도·활용시간), 학업성취(성취수준 현황·분포), 학습수행(콘텐츠 유형별 활용도·문항풀이·과제 수행도), 
맞춤학습(AI튜터·학생 주도 평가·교사 콘텐츠 재구성·교사 피드백), 교사활용지수, 일일현황의 6대 영역에 걸쳐 약 40여 개 세부 분석 화면을 구축할 것을 제안한다. 
각 화면은 다차원 필터링, 그래프-표 연계 시각화, 엑셀 내보내기, 산출식 투명 공개 등의 공통 UI/UX 기반 위에 구성되어야 하며, 이를 통해 교육 관리자가 데이터 기반의 학습 운영 의사결정을 내릴 수 있는 분석 환경을 제공하여야 한다.

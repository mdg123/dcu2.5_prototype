## 요구사항 정의

학생이 직접 만든 작품(이미지, 영상, 텍스트)을 **등록하고 전시**할 수 있는 "나도 예술가" 갤러리를 제작한다. 교사 승인 프로세스를 거쳐 전시되며, 채움클래스와 연계하여 동시 게재가 가능하다.

---

## 상세 UI/UX 요구사항

### 1. 갤러리 메인 (전시 공간)

#### 1-1. 갤러리 레이아웃
- **Masonry 그리드** (Pinterest 스타일): 다양한 크기의 작품 카드가 자연스럽게 배치
- 각 작품 카드:
  - 작품 썸네일 (이미지: 원본 비율 유지, 영상: 재생 아이콘 오버레이)
  - 작품 제목 (1줄)
  - 작성자 (학생 이름 + 학교/학년)
  - 작성일
  - ❤️ 좋아요 수 + 💬 댓글 수 + 👁️ 조회수
- 호버 효과: 오버레이 + "자세히 보기" 텍스트

#### 1-2. 필터/정렬
- 카테고리 탭: 전체 / 미술 / 음악 / 문학 / 영상 / 기타
- 정렬: 최신순 / 인기순 / 좋아요순
- 학교급 필터: 초등 / 중등 / 고등
- 검색: 제목, 작성자 검색

#### 1-3. 작품 상세 보기 (모달 또는 페이지)
```
┌──────────────────────────────────────────┐
│  [←] 나도 예술가                           │
│                                          │
│  ┌──────────────────────────────────────┐│
│  │                                      ││
│  │        작품 이미지/영상 뷰어           ││
│  │        (이미지: 확대 가능)             ││
│  │        (영상: 인라인 플레이어)          ││
│  │                                      ││
│  └──────────────────────────────────────┘│
│                                          │
│  "봄의 향기" — 수채화                      │
│  🎨 김민준 · 청주중앙초 3학년              │
│  2026.03.03                              │
│                                          │
│  작품 설명:                               │
│  봄에 피는 꽃들을 보고 느낀 감정을          │
│  수채화로 표현했습니다.                     │
│                                          │

│                                          │
│  ── 댓글 ──                              │
│  이서연: 색감이 정말 예뻐요! 🎨            │
│  박선생님: 구도가 훌륭하구나! 👏           │
│  [댓글 입력...]              [등록]        │
└──────────────────────────────────────────┘
```

### 2. 작품 등록 폼 (학생)

```
┌─────────────────────────────────────┐
│ 🎨 작품 등록하기                      │
│                                     │
│ 작품 제목: [____________________]    │
│                                     │
│ 카테고리: [미술 ▼]                   │
│                                     │
│ 작품 유형:                           │
│ ◉ 이미지  ○ 영상  ○ 텍스트          │
│                                     │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ │   📁 파일을 드래그하거나          │ │
│ │      클릭하여 업로드              │ │
│ │                                 │ │
│ │   이미지: JPG, PNG (최대 10MB)   │ │
│ │   영상: MP4 (최대 100MB)         │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 작품 설명:                           │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ ── 채움클래스 연계 ──                │
│ □ 채움클래스에도 동시 게재            │
│   클래스 선택: [초등 수학 3-1 ▼]     │
│                                     │
│ ── 태그 ──                          │
│ [수채화] [봄] [+ 태그 추가]          │
│                                     │
│       [임시저장]  [등록 신청하기]     │
└─────────────────────────────────────┘
```

- 이미지 업로드: 드래그앤드롭 + 클릭 선택, 미리보기 표시
- 영상 업로드: 파일 선택 + 업로드 진행률 표시
- 텍스트: 리치 텍스트 에디터 (간단한 서식)

### 3. 교사 승인 프로세스

#### 3-1. 승인 대기 목록 (교사 뷰)
```
┌─────────────────────────────────────────────┐
│ 📋 승인 대기 작품 (3건)                        │
│                                             │
│ ┌─── 작품 1 ────────────────────────────────┐│
│ │ [썸네일]  "봄의 향기" — 김민준 (3학년)      ││
│ │           2026.03.03 등록                  ││
│ │           카테고리: 미술                    ││
│ │                                           ││
│ │ [미리보기]  [✅ 승인]  [❌ 반려]             ││
│ └───────────────────────────────────────────┘│
│                                             │
│ ┌─── 작품 2 ────────────────────────────────┐│
│ │ ...                                       ││
│ └───────────────────────────────────────────┘│
└─────────────────────────────────────────────┘
```

#### 3-2. 반려 사유 모달
```
┌─────────────────────────────────────┐
│ ❌ 작품 반려                          │
│                                     │
│ 반려 사유:                           │
│ ○ 부적절한 내용 포함                  │
│ ○ 저작권 문제                        │
│ ○ 작품 설명 부족                     │
│ ○ 기타                              │
│                                     │
│ 상세 사유 (선택):                    │
│ [____________________________]      │
│                                     │
│          [취소]  [반려 처리]          │
└─────────────────────────────────────┘
```

#### 3-3. 작품 상태 흐름
```
[등록 신청] → [승인 대기 🟡] → [승인 🟢] → [전시 중]
                            → [반려 🔴] → [수정 후 재신청]
```

### 4. 채움클래스 연계 제출

- 작품 등록 시 "채움클래스에도 동시 게재" 체크 옵션
- 클래스 선택 드롭다운 (내가 참여 중인 클래스 목록)
- 게재 시 해당 클래스의 일반게시판에 자동 포스팅 (Mock)

## 디자인 사양

### 색상
```
갤러리 배경:     #F9FAFB
카드 배경:       #FFFFFF
오버레이:        rgba(0, 0, 0, 0.4)
좋아요 활성:     #EF4444
카테고리 칩:
  미술: #8B5CF6
  음악: #EC4899
  문학: #F59E0B
  영상: #3B82F6
  기타: #6B7280
승인 대기:       #FEF3C7
승인 완료:       #D1FAE5
반려:            #FEE2E2
```

### 레이아웃
- Masonry 그리드: CSS columns 또는 JS Masonry
- 카드: `border-radius: 12px`, `overflow: hidden`
- 상세 모달: 최대 900px
- 업로드 영역: `border: 2px dashed #D1D5DB`, `border-radius: 12px`

### 반응형
- 768px 이하: 1열 그리드
- 1024px 이하: 2열 그리드
- 1200px 이상: 3~4열 그리드

## 요구사항 정의

채움클래스 내에서 **설문조사**를 등록, 응답, 결과 확인할 수 있는 기능을 제작한다. 설문 등록 권한 설정, 응답 중 수정, 예약 게시, 설문 복사(클래스 선택), 결과 다운로드, 목록 정렬/필터링을 포함한다.

---

## 상세 UI/UX 요구사항

### 1. 설문 목록

#### 1-1. 목록 UI
- 카드형 목록:
  - 설문 제목
  - 상태 배지: 🟢 진행중 / 🟡 예약됨 / ⚪ 마감 / 📝 임시저장
  - 기간: "2026.03.04 ~ 2026.03.11"
  - 응답 현황: "응답 25/32명 (78%)" + 미니 프로그레스 바
  - 작성자
  - 액션 메뉴 (⋮): 수정 | 복사 | 삭제 | 결과 보기

#### 1-2. 필터/정렬
- 필터 탭: 전체 / 진행중 / 예약 / 마감 / 임시저장
- 정렬: 최신순 / 마감임박순 / 응답률순
- 검색: 제목 검색

#### 1-3. 등록 버튼
- "+ 설문 만들기" 버튼 (Primary)
- 권한 없으면 비활성 + 툴팁

### 2. 설문 등록/수정 폼

#### 2-1. 기본 정보
```
┌─────────────────────────────────────────┐
│ 📝 설문 만들기                            │
│                                         │
│ 제목: [____________________________]     │
│ 설명: [____________________________]     │
│                                         │
│ 기간: [2026-03-04] ~ [2026-03-11]       │
│ □ 예약 게시 (지정 시간에 자동 게시)        │
│   예약 일시: [2026-03-04 09:00]          │
│                                         │
│ 대상: ◉ 전체 멤버  ○ 특정 멤버 지정       │
│ 익명 응답: [토글 ON/OFF]                  │
│ 응답 수정 허용: [토글 ON/OFF]             │
│                                         │
│ ── 알림 설정 ──                          │
│ □ 게시 시 알림 발송                       │
│ □ 마감 1일 전 리마인더                    │
└─────────────────────────────────────────┘
```

#### 2-2. 문항 편집 영역
문항 유형 선택 후 추가:

**지원 문항 유형:**
1. **객관식 (단일 선택)**: 라디오 버튼 + 보기 목록
2. **객관식 (다중 선택)**: 체크박스 + 보기 목록
3. **주관식 (단답형)**: 텍스트 입력 (한 줄)
4. **주관식 (장문형)**: 텍스트에어리어 (여러 줄)
5. **선형 배율**: 1~5 또는 1~10 (예: 매우불만족~매우만족)
6. **날짜**: 날짜 선택기

각 문항 카드:
```
┌─────────────────────────────────────┐
│ Q1. [문항 내용 입력]          [유형 ▼] │
│ □ 필수 응답                          │
│                                     │
│  ○ 보기 1: [_____________] [×]       │
│  ○ 보기 2: [_____________] [×]       │
│  ○ 보기 3: [_____________] [×]       │
│  [+ 보기 추가]                       │
│                                     │
│  [↑][↓] 순서 이동   [복제] [삭제]     │
└─────────────────────────────────────┘
```

- "+ 문항 추가" 버튼 (문항 유형 선택 드롭다운)
- 문항 드래그앤드롭 순서 변경
- 문항 복제/삭제

#### 2-3. 하단 액션
- [임시저장] [미리보기] [게시하기]

### 3. 설문 응답 화면 (학생 뷰)

```
┌─────────────────────────────────────┐
│ 📋 수업 만족도 조사                    │
│ 기간: 2026.03.04 ~ 2026.03.11       │
│ 익명 응답                            │
│                                     │
│ Q1. 이번 수업은 이해하기 쉬웠나요?     │
│  ○ 매우 그렇다                       │
│  ● 그렇다                           │
│  ○ 보통이다                          │
│  ○ 그렇지 않다                       │
│  ○ 매우 그렇지 않다                   │
│                                     │
│ Q2. 개선이 필요한 점을 적어주세요.     │
│ ┌─────────────────────────────────┐ │
│ │ 실험 시간이 더 있으면 좋겠습...   │ │
│ └─────────────────────────────────┘ │
│                                     │
│          [임시저장]  [제출하기]        │
└─────────────────────────────────────┘
```

- 응답 중 수정: 이미 선택한 답변 변경 가능
- 제출 전 확인 모달: "제출하시겠습니까? 제출 후 수정이 불가합니다." (수정 허용 설정에 따라 문구 변경)

### 4. 결과 조회 (교사 뷰)

#### 4-1. 결과 요약
- 응답률: 원형 게이지 "25/32명 (78%)"
- 문항별 결과:
  - 객관식: 가로 바 차트 (각 보기별 응답 비율)
  - 선형 배율: 평균값 + 분포 바
  - 주관식: 응답 목록 (스크롤)

#### 4-2. 멤버별 결과
- 테이블: 멤버명 | 응답일 | 응답 상태 (응답/미응답)
- 개별 응답 상세 보기 모달

#### 4-3. 다운로드
- "결과 다운로드" 버튼 → 형식 선택: Excel (.xlsx) / CSV

### 5. 설문 복사

```
┌─────────────────────────────────────┐
│ 📋 설문 복사                          │
│                                     │
│ 복사할 클래스 선택:                    │
│ ┌─────────────────────────────────┐ │
│ │ □ 초등 수학 3-1 (나의 클래스)    │ │
│ │ ☑ 중학 과학 실험교실             │ │
│ │ □ 고등 영어 독해 마스터          │ │
│ └─────────────────────────────────┘ │
│                                     │
│ ☑ 문항 복사  ☑ 설정 복사             │
│ □ 기간 유지 (해제 시 새로 설정)       │
│                                     │
│           [취소]  [복사하기]          │
└─────────────────────────────────────┘
```

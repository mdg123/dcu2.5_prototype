## 요구사항 정의

채움클래스 내에서 자유롭게 글을 작성하고 소통할 수 있는 **일반게시판** 기능을 신규 제작한다. 일반형/갤러리형 두 가지 유형을 지원하며, 승인 설정, 읽기/쓰기 권한 분리, 댓글에 이미지 드래그앤드롭, 동영상 임베딩 기능을 포함한다.

---

## 상세 UI/UX 요구사항

### 1. 게시판 목록

#### 1-1. 보기 모드 전환
- 상단 우측에 토글 버튼: **일반형** (리스트) / **갤러리형** (그리드)

**일반형 (리스트 뷰):**
- 테이블 형태: 번호 | 제목 | 작성자 | 날짜 | 조회 | 좋아요 | 댓글수
- 공지 게시물: 상단 고정, 배경색 `#FFFBEB`, "📌 공지" 라벨
- 행 호버 효과: 배경 `#F9FAFB`
- 제목에 첨부파일 아이콘 📎, 이미지 있으면 🖼️ 표시

**갤러리형 (그리드 뷰):**
- 카드 그리드: 3열 (데스크탑), 2열 (태블릿), 1열 (모바일)
- 각 카드: 썸네일 이미지 (첫 번째 이미지 또는 기본 아이콘) + 제목 + 작성자 + 날짜

#### 1-2. 검색 및 필터
- 검색바: "제목, 내용, 작성자 검색"
- 필터 드롭다운: 전체 / 공지 / 일반 / 내 글
- 정렬: 최신순 / 조회순 / 좋아요순 / 댓글순

#### 1-3. 글쓰기 버튼
- 우측 상단 Primary 블루 버튼: "+ 글쓰기"
- 권한 없으면 비활성(disabled) + 툴팁 "글쓰기 권한이 없습니다"

### 2. 글쓰기/수정 폼

#### 2-1. 폼 구성
```
┌─────────────────────────────────────┐
│ 📝 게시글 작성                        │
│                                     │
│ 제목: [________________________]     │
│                                     │
│ 유형: ◉ 일반  ○ 공지 (개설자만)       │
│                                     │
│ ┌─────────────────────────────────┐ │
│ │ WYSIWYG 에디터                   │ │
│ │ [B][I][U] [H1][H2] [목록] [링크] │ │
│ │ [이미지][동영상][파일첨부]         │ │
│ │                                 │ │
│ │                                 │ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 첨부파일: [파일 선택] (최대 5개, 20MB) │
│ ┌─ file1.pdf (2.3MB) [×]           │
│ └─ image.png (1.1MB) [×]           │
│                                     │
│       [취소]  [미리보기]  [등록하기]   │
└─────────────────────────────────────┘
```

#### 2-2. 미디어 삽입
- **이미지**: 에디터 내 드래그앤드롭 또는 버튼 클릭 삽입
- **동영상**: YouTube/Vimeo URL 입력 → 임베디드 플레이어로 인라인 표시
- **첨부파일**: 파일 선택 다이얼로그, 진행률 표시, 파일 크기/형식 표시

### 3. 게시글 상세 보기

#### 3-1. 게시글 본문
- 제목, 작성자(프로필 아바타 + 이름), 작성일, 조회수
- 본문 (HTML 렌더링, 이미지 자동 스케일링, 동영상 임베디드)
- 첨부파일 목록 (다운로드 링크)
- 좋아요 버튼 ❤️ + 카운트
- 작성자에게: 수정 | 삭제 버튼

#### 3-2. 댓글 영역
- 댓글 목록: 작성자 아바타 + 이름 + 시간 + 내용
- **이미지 댓글**: 댓글 입력란에 이미지 드래그앤드롭 또는 클립보드 붙여넣기 지원
- 댓글 입력: 텍스트 영역 + 이미지 첨부 버튼 + "등록" 버튼
- 대댓글 (1단계 답글)

### 4. 승인 설정 (개설자 전용)

#### 4-1. 게시판 설정 패널
```
┌─────────────────────────────────────┐
│ ⚙️ 게시판 설정                        │
│                                     │
│ 게시판 유형: ◉ 일반형  ○ 갤러리형     │
│                                     │
│ ── 승인 설정 ──                      │
│ 글 등록 승인: [토글 ON/OFF]           │
│  └ ON 시: 멤버 글 → 대기 → 승인/반려  │
│                                     │
│ ── 권한 설정 ──                      │
│ 읽기 권한: ◉ 전체 멤버  ○ 개설자만    │
│ 쓰기 권한: ◉ 전체 멤버  ○ 개설자만    │
│ 댓글 권한: ◉ 전체 멤버  ○ 개설자만    │
│                                     │
│              [저장]                   │
└─────────────────────────────────────┘
```

#### 4-2. 승인 관리 UI
- 승인 대기 목록: 게시글 제목 + 작성자 + 작성일 + [승인] [반려] 버튼
- 상태 배지: 🟡 대기 / 🟢 승인 / 🔴 반려
- 반려 시 사유 입력 모달
- 일괄 승인 체크박스 + "선택 승인" 버튼

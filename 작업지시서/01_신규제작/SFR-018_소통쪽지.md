## 요구사항 정의

클래스 내에서 개설자와 멤버, 또는 멤버 간에 **쪽지(메시지)** 를 주고받을 수 있는 소통쪽지 기능을 제작한다. 양방향 채팅형과 단방향 쪽지형 두 유형을 지원하며, 사용 범위를 개설자가 설정할 수 있다.

---

## 상세 UI/UX 요구사항

### 1. 소통쪽지 메인 레이아웃

2패널 레이아웃 (데스크탑):

```
┌────────────────┬──────────────────────────┐
│  대화 목록       │  대화 내용                 │
│                │                          │
│ 🔍 검색         │  ┌──── 김영희 선생님 ────┐ │
│                │  │                      │ │
│ ┌────────────┐ │  │ 안녕하세요, 수학      │ │
│ │ 김영희 선생님│ │  │ 과제 관련 질문이     │ │
│ │ 과제 관련..  │ │  │ 있습니다.            │ │
│ │ 오후 2:30   │ │  │          오후 2:25  │ │
│ ├────────────┤ │  │                      │ │
│ │ 박민준      │ │  │  네, 어떤 부분이     │ │
│ │ 감사합니다!  │ │  │  궁금한가요?         │ │
│ │ 오전 11:20  │ │  │ 오후 2:30            │ │
│ ├────────────┤ │  │                      │ │
│ │ ...        │ │  └──────────────────────┘ │
│ └────────────┘ │                          │
│                │  ┌──────────────────────┐ │
│                │  │ 메시지 입력...    [전송]│ │
│                │  └──────────────────────┘ │
└────────────────┴──────────────────────────┘
```

### 2. 대화 목록 (좌측 패널)

- 검색바: 이름 또는 내용으로 검색
- 대화 항목:
  - 상대방 프로필 아바타 (40px 원형)
  - 상대방 이름 + 역할 배지 (선생님/학생)
  - 마지막 메시지 미리보기 (1줄, 말줄임)
  - 시간 표시 (오늘: "오후 2:30", 어제 이전: "3/3")
  - 안 읽은 메시지 수 배지 (빨간 원형)
- 정렬: 최신 대화순
- "새 쪽지" 버튼 (상단)

### 3. 대화 내용 (우측 패널)

#### 3-1. 양방향 채팅형
- 카카오톡/iMessage 스타일 말풍선 UI
- 내 메시지: 우측 정렬, 블루 배경 (#2563EB), 흰 텍스트
- 상대 메시지: 좌측 정렬, 회색 배경 (#F3F4F6), 검정 텍스트
- 시간 표시: 말풍선 아래, 작은 글씨
- 날짜 구분선: "─── 2026년 3월 4일 ───"
- 읽음 표시: ✓ (전송됨) / ✓✓ (읽음)

#### 3-2. 단방향 쪽지형
- 편지 형식 UI
- 보낸 쪽지 / 받은 쪽지 탭 구분
- 쪽지 카드: 보낸이 | 제목 | 날짜 | 읽음 상태
- 상세 보기: 편지지 스타일 레이아웃

#### 3-3. 메시지 입력
- 텍스트 입력 영역 (자동 높이 조절, 최대 5줄)
- 전송 버튼 (Enter 키로도 전송, Shift+Enter 줄바꿈)
- 이미지 첨부 버튼 (선택적)
- 파일 첨부 버튼 (선택적)

### 4. 사용 범위 설정 (개설자)

#### 4-1. 설정 패널
```
┌─────────────────────────────────────┐
│ ⚙️ 소통쪽지 설정                      │
│                                     │
│ ── 사용 범위 ──                      │
│ ◉ 개설자 ↔ 멤버 (개별 상담용)         │
│ ○ 멤버 간 허용 (자유 소통)            │
│ ○ 비활성화 (소통쪽지 사용 안 함)       │
│                                     │
│ ── 메시지 유형 ──                     │
│ ◉ 양방향 채팅 (실시간 대화형)          │
│ ○ 단방향 쪽지 (편지형)                │
│                                     │
│ ── 알림 설정 ──                      │
│ 새 메시지 알림: [토글 ON]             │
│                                     │
│              [저장]                   │
└─────────────────────────────────────┘
```

### 5. 새 쪽지 작성 모달

```
┌─────────────────────────────────────┐
│ ✉️ 새 쪽지 보내기                     │
│                                     │
│ 받는 사람: [멤버 검색/선택 ▼]         │
│ ┌─ 김민준 ×  이서연 ×                │
│                                     │
│ 메시지:                              │
│ ┌─────────────────────────────────┐ │
│ │                                 │ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                                     │
│            [취소]  [보내기]           │
└─────────────────────────────────────┘
```

## 디자인 사양

### 색상
```
내 말풍선:      #2563EB 배경, #FFFFFF 텍스트
상대 말풍선:    #F3F4F6 배경, #1A1A2E 텍스트
안읽음 배지:    #EF4444
읽음 표시:      #10B981
날짜 구분선:    #9CA3AF
대화 목록 선택: #EFF6FF
호버:           #F9FAFB
```

### 레이아웃
- 좌측 패널: 너비 320px (고정)
- 우측 패널: 나머지 영역 (flex-grow)
- 말풍선: `border-radius: 16px`, `padding: 10px 16px`, `max-width: 70%`
- 모바일: 대화 목록만 표시 → 클릭 시 대화 내용 전체 화면

### 반응형
- 768px 이하: 1패널 모드 (목록↔대화 전환)
- 1024px 이하: 좌측 패널 240px 축소

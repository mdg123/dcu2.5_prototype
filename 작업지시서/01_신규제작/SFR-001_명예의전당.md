## 요구사항 정의

다채움 포털에 **인기 클래스 및 인기 크리에이터 랭킹**을 보여주는 "명예의 전당" 전용 페이지를 신규 제작한다. 포털 메인에는 이 페이지로 연결되는 위젯을 배치한다.

---

## 상세 UI/UX 요구사항

### 1. 명예의 전당 전용 페이지 (`hall-of-fame.html`)

#### 1-1. 페이지 헤더
- 페이지 타이틀: "명예의 전당" (아이콘: 트로피 🏆 또는 Font Awesome `fa-trophy`)
- 기간 선택 드롭다운: "2026년 3월" 형태 (월간 리스트)
- 탭 전환: **인기 클래스** / **인기 크리에이터**

#### 1-2. 인기 클래스 TOP 10~100 (월간 리스트)
- **랭킹 카드 리스트** 형태로 표시
- 각 카드에 포함할 정보:
  - 순위 번호 (1~3위는 금/은/동 메달 아이콘으로 강조)
  - 클래스 썸네일 이미지 (64x64 라운드)
  - 클래스명 (링크 → 클래스 상세)
  - 개설자 이름
  - 멤버 수
  - 선정 기준 점수 표시 (활동량/출석률/피드백 수/멤버 수 4가지)
- **선정 기준 시각화**:
  - 각 기준별 점수를 수평 바 또는 레이더 차트로 표시
  - 기준: 활동량, 출석률, 피드백 수, 멤버 수
- 정렬 옵션: 종합 점수순 / 활동량순 / 출석률순 / 피드백순 / 멤버수순
- 표시 개수 조절: TOP 10 / TOP 30 / TOP 50 / TOP 100

#### 1-3. 인기 크리에이터 TOP 10~100
- 크리에이터 카드:
  - 순위 (1~3위 메달)
  - 프로필 이미지 (원형 아바타)
  - 크리에이터 닉네임
  - 소속 학교/기관
  - 운영 클래스 수
  - 총 콘텐츠 수
  - 총 구독자 수
  - 선정 점수
- 동일한 정렬/표시 개수 옵션

#### 1-4. 선정 이력 관리 UI
- "이전 기록 보기" 버튼 → 월별 아카이브 리스트
- 이력 항목: 월/년, 선정 클래스/크리에이터 수, "상세 보기" 링크
- **명예의 전당 배지** 표시:
  - 선정 횟수에 따른 등급 배지 (브론즈 1회 / 실버 3회 / 골드 5회 / 다이아 10회)
  - 배지는 클래스명/크리에이터명 옆에 표시

### 2. 포털 메인 위젯 (포털 메인에 추가)

> ⚠️ **별도 작업**: 포털 메인(`dacheum-portal-v4 (4).html`)에 위젯을 추가하는 것은 별도 지시서로 진행. 여기서는 위젯 디자인 사양만 기술.

- 위젯 크기: 카드형 (폭 100%, 높이 자동)
- 표시 내용: TOP 3 인기 클래스 + TOP 3 인기 크리에이터 (미니 카드)
- "더보기" 링크 → 명예의 전당 페이지로 이동

## 디자인 사양

### 색상
```
메달 골드:     #FFD700
메달 실버:     #C0C0C0
메달 브론즈:   #CD7F32
랭킹 배경:     #FFFBEB (1~3위 하이라이트)
Primary 블루:  #2563eb (버튼, 링크)
카드 배경:     #FFFFFF
페이지 배경:   #F5F7FA
텍스트:        #1A1A2E (제목), #6B7280 (보조)
```

### 레이아웃
- 최대 너비: 1200px, 중앙 정렬
- 카드 그리드: 1열 (모바일) / 2열 (태블릿) / 1열 리스트형 (데스크탑)
- 탭 바: 수평, 활성 탭 하단 블루 언더라인
- 카드: `border-radius: 12px`, `box-shadow: 0 2px 8px rgba(0,0,0,0.08)`

### 폰트
- 순위 번호: Pretendard 28px Bold
- 클래스/크리에이터명: Pretendard 16px SemiBold
- 보조 텍스트: Pretendard 13px Regular, color #6B7280

### 반응형
- 768px 이하: 카드 1열, 점수 바 축소
- 1024px 이하: 2열 그리드

## 요구사항 정의

수업, 과제, 평가, 알림장 등 **클래스 내에서 대상 멤버를 선택**해야 하는 모든 기능에서 공통으로 사용할 **멤버 지정 UI 컴포넌트**를 제작한다. "전체 대상" 또는 "특정 멤버 대상" 선택, 멤버 검색/다중 선택, 변경 확인 모달 등을 포함한다.

---

## 상세 UI/UX 요구사항

### 1. 대상 선택 영역

#### 1-1. 대상 유형 라디오 버튼
```
◉ 전체 멤버 (32명)
○ 특정 멤버 지정
```
- "전체 멤버" 선택 시: 별도 선택 UI 없이 전체 자동 적용
- "특정 멤버 지정" 선택 시: 아래 멤버 선택 패널 펼침 (슬라이드 다운 애니메이션)

#### 1-2. 멤버 선택 패널 (특정 멤버 모드)

**검색 바:**
- 플레이스홀더: "이름, 학번으로 검색"
- 실시간 필터링 (키 입력 시 즉시 반영)
- 검색 아이콘 (좌측) + 클리어(×) 버튼 (우측)

**멤버 목록 (좌측 패널):**
- 체크박스 + 프로필 아바타(32px 원형) + 이름 + 학번/역할
- "전체 선택" 체크박스 (상단)
- 선택된 멤버 수 표시: "3/32명 선택됨"
- 스크롤 가능 (max-height: 300px, overflow-y: auto)
- 체크된 항목 배경 하이라이트 (#EFF6FF)

**선택된 멤버 영역 (우측 또는 하단):**
- 선택된 멤버를 **칩(Chip/Tag)** 형태로 표시
- 각 칩: 이름 + 제거(×) 버튼
- 빈 상태: "선택된 멤버가 없습니다" 안내 문구

#### 1-3. 그룹 선택 (선택적)
- "그룹으로 선택" 토글
- 미리 정의된 그룹 목록 (1모둠, 2모둠 등)
- 그룹 선택 시 해당 그룹 멤버 일괄 체크

### 2. 확인 모달 (대상 변경 시)

#### 시나리오별 확인 모달:

**시나리오 A: 전체 → 특정 멤버로 변경**
```
┌─────────────────────────────────┐
│  ⚠️ 대상 변경 확인                │
│                                 │
│  전체 멤버 대상에서 특정 멤버      │
│  대상으로 변경하시겠습니까?        │
│                                 │
│  변경 시 기존 전체 대상 설정이     │
│  해제됩니다.                     │
│                                 │
│       [취소]  [변경하기]          │
└─────────────────────────────────┘
```

**시나리오 B: 특정 멤버 → 전체로 변경**
```
┌─────────────────────────────────┐
│  ⚠️ 대상 변경 확인                │
│                                 │
│  선택한 멤버(5명)를 해제하고       │
│  전체 멤버로 변경하시겠습니까?     │
│                                 │
│       [취소]  [변경하기]          │
└─────────────────────────────────┘
```

**시나리오 C: 이미 열람/제출 진행 중인 경우**
```
┌─────────────────────────────────┐
│  ⚠️ 진행 중 대상 변경              │
│                                 │
│  이미 3명이 열람하였습니다.        │
│  대상을 변경하면 새로 추가된       │
│  멤버에게 알림이 발송됩니다.       │
│                                 │
│  • 기존 열람자: 영향 없음         │
│  • 새 추가 멤버: 알림 발송        │
│  • 제외 멤버: 접근 제한           │
│                                 │
│       [취소]  [변경하기]          │
└─────────────────────────────────┘
```

### 3. 열람/제출 현황 모니터링

- 대상 지정 후 현황 표시 영역:
  - 프로그레스 바: "열람 25/32명 (78%)" 또는 "제출 20/32명 (63%)"
  - 상태별 필터 탭: 전체 / 완료 / 미완료
  - 멤버별 상태 아이콘: ✅ 완료 / ⏳ 진행중 / ❌ 미참여
  - 리마인더 발송 버튼: "미참여 멤버에게 알림 보내기"

## 디자인 사양

### 색상
```
선택 하이라이트:  #EFF6FF (배경), #2563eb (체크박스)
칩 배경:          #DBEAFE
칩 텍스트:        #1D4ED8
모달 오버레이:    rgba(0, 0, 0, 0.5)
경고 아이콘:      #F59E0B
완료 아이콘:      #10B981
미참여 아이콘:    #EF4444
```

### 레이아웃
- 멤버 선택 패널: 2열 (좌: 목록, 우: 선택 결과) 또는 1열 (모바일)
- 모달: 최대 480px, 중앙 정렬, `border-radius: 16px`
- 칩: `border-radius: 20px`, `padding: 4px 12px`

### 반응형
- 768px 이하: 2열 → 1열, 선택 결과 하단 배치
- 검색바 풀 너비

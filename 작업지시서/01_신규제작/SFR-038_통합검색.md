## 요구사항 정의

다채움 전체 서비스에서 사용하는 **통합 검색 오버레이 모달**을 제작한다. 새 창이 아닌 현재 화면 위에 오버레이로 표시되며, 실시간 필터링, 검색어 편의 기능(최근/인기/추천), 학교급별 배지, 키보드 탐색을 지원한다.

---

## 상세 UI/UX 요구사항

### 1. 검색 오버레이 모달

#### 1-1. 모달 열기/닫기
- **열기**: GNB 검색 아이콘(🔍) 클릭 또는 **Ctrl+K** (Mac: ⌘+K) 단축키
- **닫기**: ESC 키 / 배경(오버레이) 클릭 / 닫기(×) 버튼
- **배경**: 반투명 오버레이 `rgba(0, 0, 0, 0.5)`, backdrop-filter: blur(4px)

#### 1-2. 모달 헤더
- 모달 상단 바를 **드래그하여 위치 이동** 가능
- 드래그 핸들: 상단 중앙 가로 바 (6px × 40px, 회색)
- 최소/최대 크기 제한 (min: 480px, max: 900px 너비)

#### 1-3. 검색 입력 영역
```
┌─────────────────────────────────────────────────┐
│ ═══ (드래그 핸들)                                 │
│                                                 │
│ 🔍 [통합 검색어를 입력하세요...         ] [검색]  │
│                                                 │
│ ── 검색어 편의 기능 ──                           │
│                                                 │
│ 최근 검색어:                                     │
│ [곱셈구구 ×] [과학실험 ×] [영어 독해 ×]          │
│                                                 │
│ 인기 검색어:                                     │
│ 1. AI 학습   2. 코딩교육  3. 수학익힘            │
│ 4. 독서록    5. 실험보고서                       │
│                                                 │
│ 추천 검색어:                                     │
│ [3학년 수학] [과학 실험] [영어 회화]              │
└─────────────────────────────────────────────────┘
```

### 2. 실시간 검색 결과

#### 2-1. 입력 시 즉시 결과 표시
- 2글자 이상 입력 시 실시간 필터링 시작
- 디바운스: 300ms
- 로딩 스피너 (결과 로딩 중)

#### 2-2. 결과 영역 레이아웃
```
┌─────────────────────────────────────────────────┐
│ 🔍 [곱셈                                ] [검색] │
│                                                 │
│ ── 필터 ──                                      │
│ [전체] [수업] [콘텐츠] [과제] [평가] [게시물]      │
│                                                 │
│ 검색 결과 (23건)                                 │
│                                                 │
│ ┌───────────────────────────────────────────┐   │
│ │ 📚 3-1 곱셈구구 기초 학습        초등 🟢    │   │
│ │ 채움클래스 > 수업  |  김영희 선생님         │   │
│ │ "곱셈"구구 2단부터 9단까지...              │   │ ← 활성 (키보드 하이라이트)
│ └───────────────────────────────────────────┘   │
│ ┌───────────────────────────────────────────┐   │
│ │ 📝 곱셈 응용문제 50선             초등 🟢    │   │
│ │ 채움콘텐츠 > 문서  |  수학교육연구회        │   │
│ │ "곱셈" 응용 문제를 단계별로...             │   │
│ └───────────────────────────────────────────┘   │
│ ┌───────────────────────────────────────────┐   │
│ │ 🎯 곱셈구구 평가지                중등 🔵    │   │
│ │ 채움CBT > 평가지  |  충북교육청             │   │
│ └───────────────────────────────────────────┘   │
│                                                 │
│ ↑↓ 방향키로 탐색  Enter 선택  Esc 닫기           │
└─────────────────────────────────────────────────┘
```

#### 2-3. 검색 결과 항목 구성
- 아이콘: 유형별 (📚수업 / 📝콘텐츠 / ✏️과제 / 🎯평가 / 💬게시물)
- 제목: 검색어 하이라이트 (bold + 노란 배경)
- **학교급별 배지**: 초등(🟢 녹색) / 중등(🔵 파랑) / 고등(🟣 보라)
- 출처: "서비스 > 유형 | 작성자"
- 미리보기: 본문 일부 (검색어 포함 문장, 최대 2줄)

### 3. 상세 키워드 검색 및 조합

#### 3-1. 콘텐츠 유형별 필터링 탭
- 전체 / 수업 / 콘텐츠 / 과제 / 평가 / 게시물
- 각 탭에 검색 결과 건수 표시: "수업 (5)"
- 탭 전환 시 결과 즉시 필터링

#### 3-2. 고급 검색 옵션 (접기/펼치기)
```
┌─────────────────────────────────────┐
│ ▼ 상세 검색 옵션                     │
│                                     │
│ 학교급: □ 초등  □ 중등  □ 고등       │
│ 교과:   [전체 ▼]                    │
│ 기간:   [전체 ▼] 또는 직접 입력      │
│ 작성자: [__________]                │
│                                     │
│         [검색 조건 초기화]            │
└─────────────────────────────────────┘
```

### 4. 키보드 탐색

- **↑ / ↓ 방향키**: 검색 결과 항목 탐색 (포커스 이동)
- **Enter**: 현재 포커스된 항목 선택 (해당 페이지로 이동)
- **Esc**: 모달 닫기
- **Tab**: 필터 탭 간 이동
- 포커스된 항목: 배경색 하이라이트 (#EFF6FF) + 좌측 블루 보더

### 5. 빈 상태 / 에러 상태

```
검색 결과 없음:
┌─────────────────────────────────────┐
│  🔍                                  │
│  "곱쌤구구"에 대한 검색 결과가        │
│  없습니다.                           │
│                                     │
│  혹시 이것을 찾으셨나요?             │
│  [곱셈구구] [곱셈] [구구단]          │
└─────────────────────────────────────┘
```

## 디자인 사양

### 색상
```
오버레이 배경:     rgba(0, 0, 0, 0.5)
모달 배경:         #FFFFFF
검색 입력 테두리:   #2563EB (포커스 시)
검색어 하이라이트:  #FEF3C7 배경, #92400E 텍스트
포커스 항목:       #EFF6FF 배경, 좌측 4px #2563EB 보더
학교급 배지:
  초등: #D1FAE5 배경, #059669 텍스트
  중등: #DBEAFE 배경, #2563EB 텍스트
  고등: #EDE9FE 배경, #7C3AED 텍스트
최근 검색어 칩:    #F3F4F6 배경
인기 검색어:       #1A1A2E 텍스트
추천 검색어 칩:    #EFF6FF 배경, #2563EB 텍스트
```

### 레이아웃
- 모달: 기본 너비 640px, 최대 높이 80vh
- 결과 영역: `overflow-y: auto`, 스크롤
- `border-radius: 16px`
- `box-shadow: 0 20px 60px rgba(0, 0, 0, 0.3)`
- 초기 위치: 화면 상단 1/4 지점 중앙
- 드래그 시: `position: fixed` 유지

### 반응형
- 768px 이하: 모달 전체 화면 (fullscreen), 드래그 비활성
- 1024px 이하: 모달 너비 90%

## 재사용 안내

이 컴포넌트는 **독립 데모 페이지**로 제작하되, 핵심 HTML/CSS/JS를 **common-nav.js에 통합**하여 모든 프로토타입에서 검색 아이콘 클릭 시 동일한 검색 오버레이가 표시되도록 할 수 있게 설계할 것. 통합 작업은 별도 지시서로 진행.

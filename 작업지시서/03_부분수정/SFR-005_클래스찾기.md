## 추가/보완 항목

### 1. 코드 검색 UX 강화
**현재 상태**: 사용자가 코드 입력 후 "검색" 버튼 클릭
**개선 사항**:
- 입력 중 실시간 유효성 검증 (정규식: 4-6자 영문/숫자)
- 입력 완료 시 자동으로 검색 실행
- 유효하지 않은 형식 시 에러 메시지 표시 ("올바른 코드 형식이 아닙니다")
- 검색 결과 없음 → "일치하는 클래스가 없습니다" 안내 문구 추가

### 2. 필터 옵션 확장
**자동승인 필터**:
- 기존: 토글 (On/Off)
- 추가: "관심 없음" 필터 추가 (신청승인 필요 클래스 중심 표시)

**학교급 필터 추가**:
- 선택지: 전체, 초등학교, 중학교, 고등학교
- 필터 UI: 드롭다운 또는 칩 토글 형식

### 3. 클래스 상세 모달 정보 보강
**추가 정보 섹션**:
- 운영 기간: "2024.01 ~ 2024.12" (시각적 인디케이터: 진행중/종료)
- 활동 현황: 누적 활동 통계 (최근 7일/30일 가입자 수)
- 주관 기관: 학교/교육청명
- 추천율: ⭐ 평점 + 리뷰 수 (e.g., "4.5 / 5 (123명)")

### 4. 맞춤 추천 알고리즘 결과 UI 분리
**현재**: AI 추천 섹션이 고정된 위치에 5개 수평 스크롤
**개선**:
- AI 추천 헤더에 뱃지 추가: "당신을 위한 추천" (아이콘 + 텍스트)
- 추천 알고리즘 출처 표시 (e.g., "학습 이력 기반", "인기도순" 등)
- 추천 카드에 미니 배지: "당신을 위한" 라벨 추가
- 추천 항목 클릭 시 분석 모달: "추천 이유" 간단 설명

## 추가/보완 항목

### 1. 지능형 검색 (성취기준/키워드 통합)
**현재**: 단순 키워드 검색만 가능
**개선**:

#### 1-1. 검색창 고도화
- 위치: 공개콘텐츠 탭 상단
- 입력창: placeholder "콘텐츠, 성취기준, 크리에이터 검색"
- 검색 버튼: 우측 또는 Enter 키로 검색
- 검색 옵션 토글: 우측에 "🔧 고급검색" 버튼

#### 1-2. 고급 검색 모달
**팝업 위치**: 검색창 아래 또는 별도 모달

**검색 필드**:
1. **키워드 검색** (기본)
   - 입력창: "수학 기하학" (여러 키워드 AND 연산)

2. **성취기준 검색** (신규)
   - 선택 방식: 트리뷰 또는 칩 선택
   - 구조: 과목 → 학년 → 단원 → 성취기준
   - 예: [수학] > [중학교 1학년] > [기하] > [도형의 성질]
   - 다중 선택 가능 (또는 관계)
   - 검색 결과: "성취기준 '도형의 성질'과 관련된 콘텐츠 45개"

3. **크리에이터 검색** (신규)
   - 검색 상자: "크리에이터명 또는 채널 검색"
   - 자동완성: 기존 크리에이터 목록 제시
   - 클릭 → 선택된 크리에이터 콘텐츠만 필터링

4. **콘텐츠 유형** (신규)
   - 체크박스: 수업, 과제, 평가, 자료 (다중 선택)

5. **학급/학년** (신규)
   - 드롭다운: 초등학교 / 중학교 / 고등학교 (또는 구체적 학년)

6. **정렬순서** (신규)
   - 라디오 버튼:
     - 관련성순 (기본)
     - 조회수순 (인기순)
     - 좋아요순
     - 최신순
     - 평점순

**버튼**:
- 하단 "검색" 버튼 (Primary 블루)
- "초기화" 버튼 (Ghost)

#### 1-3. 검색 결과 요약
- 검색창 아래: "키워드 'python' + 성취기준 '데이터 처리' 관련 콘텐츠 23개"
- 검색 조건 칩: 제거 가능한 태그 형식으로 표시

### 2. 3종 보기 모드 (카드형/리스트형/피드형)
**위치**: 콘텐츠 목록 우측 상단
**요소**:
- 버튼 3개: "🔲 카드형 | ☰ 리스트형 | 📱 피드형"
- 선택된 모드: 배경 #2563eb, 텍스트 white
- 미선택: 배경 #f3f4f6, 텍스트 #6B7280

#### 2-1. 카드형 뷰
- 레이아웃: 그리드 3~4열 (반응형)
- 각 카드:
  - 썸네일 (상단, 240px × 180px)
  - 제목 (13px, weight 600, 최대 2줄)
  - 크리에이터명 (11px, #6B7280) + 채널 배지 (tiny)
  - 설명 (12px, 1줄)
  - 하단: 소셜 지표 + "담기" 버튼
- 호버: 그림자 강화, 스케일 1.02

#### 2-2. 리스트형 뷰
- 테이블 구조:
  - 체크박스 | 썸네일(mini) | 제목 | 크리에이터 | 유형 | 조회수 | 좋아요 | 평점 | 담기
- 행 호버: 배경 #f9fafb
- 정렬 가능: 각 열 헤더 클릭

#### 2-3. 피드형 뷰 (신규, SNS 스타일)
- 레이아웃: 전체 너비 카드, 수직 스크롤
- 각 카드 (height ~400px):
  - 상단: 크리에이터 프로필 + 팔로우 버튼
  - 썸네일 (전체 너비, 240px height)
  - 제목 (14px, weight 600)
  - 설명 (13px, 최대 3줄)
  - 성취기준 태그 (여러 개)
  - 하단: 소셜 지표 (조회수/좋아요/댓글) + "바로 풀기" 버튼 + 공유 메뉴
- 호버: 배경 미세하게 변경
- 무한 스크롤 또는 "더보기" 버튼

### 3. 소셜 지표 표시
**표시 위치**: 각 카드 하단 (또는 피드형에서 하단)

**지표 요소**:
1. **조회수**: 👁️ 아이콘 + "1.2K" (또는 "1,234")
2. **좋아요/별점**: ⭐ 아이콘 + "4.8" (또는 ❤️ + "234")
3. **댓글 수**: 💬 아이콘 + "12"
4. **저장/담기**: 📌 아이콘 + "56"

**스타일**:
- 크기: 11px, weight 500, color #6B7280
- 아이콘 + 수치 간격: 4px
- 각 지표 간 간격: 12px

**동작**:
- 조회수/좋아요 클릭: 상세 정보 팝오버 (또는 무시)
- 댓글 수 클릭: 댓글 섹션으로 스크롤

### 4. 원클릭 담기 버튼
**위치**: 각 카드 우측 하단 (또는 호버 시 표시)
**요소**:
- 버튼: "📌 담기" 또는 단순 아이콘
- 스타일: Primary 블루 배경, 흰색 텍스트, padding 6px 12px, border-radius 4px
- 호버: 배경 진하게 (#1d4ed8)

**동작**:
1. 클릭 시 폴더 선택 모달 열기:
   ```
   어느 폴더에 담으시겠어요?
   ☐ 내 폴더 1
   ☐ 내 폴더 2
   ☐ 새 폴더 만들기
   ```
2. 선택 후 "담기" 버튼
3. 완료 토스트: "콘텐츠가 [폴더명]에 저장되었습니다" (우측 하단, 3초)
4. 버튼 상태 변경: "✓ 담아짐" (배경 회색, 클릭 불가 또는 취소 기능)

### 5. 콘텐츠 세트화 UI
**위치**: 검색 결과 상단 또는 별도 섹션
**요소**:
- 섹션 제목: "추천 콘텐츠 세트" (13px, weight 600)
- 설명: "성취기준과 함께 학습할 수 있는 콘텐츠 모음입니다" (11px, #6B7280)

**세트 카드**:
- 제목: "기하학 완벽 가이드" (14px, weight 600)
- 포함 콘텐츠: "15개 콘텐츠 모음" (12px, #6B7280)
- 총 학습시간: "약 12시간" (11px)
- 세트 커버: 썸네일 3개 겹침 (또는 세트 대표 이미지)
- 버튼: "세트 전체 담기" (또는 "시작하기")
- 호버: 그림자 강화

### 6. 바로 풀기 버튼
**위치**: 각 카드 우측 하단 또는 "담기" 버튼 옆
**요소**:
- 버튼: "► 바로 풀기" 또는 "▶ 풀이하기"
- 스타일: Secondary 회색 배경, 텍스트 #1f2937, padding 6px 12px
- 호버: 배경 #e5e7eb

**동작**:
- 클릭 시 콘텐츠 풀이 인터페이스로 이동 (또는 모달 오픈)
- 새 탭에서 열 수도 있음

## 주의사항

1. **SPA 네비게이션 유지**: `#public` 해시 라우팅 변경 금지
2. **성능**: 검색 결과가 1000개 이상일 경우 페이지네이션 또는 무한스크롤 필수
3. **더미 데이터**: 성취기준, 크리에이터, 소셜 지표 데이터가 일관성 있어야 함
4. **접근성**: 보기 모드 버튼은 충분한 크기 (최소 32px height)
5. **모달**: 폴더 선택 모달은 최대 스크롤 가능 (항목 많음)
6. **애니메이션**: 토스트 Fade In/Out 부드럽게 (300ms)

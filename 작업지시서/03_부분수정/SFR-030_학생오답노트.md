| 항목 | 내용 |
|------|------|

| **현재상태** | 교사/학생 뷰 전환, 4개 탭, 요약카드, Chart.js 차트, 취약영역 분석, 문항 리스트 |

## 기존 구현 현황

### 현재 보유 기능
- **교사/학생 뷰 전환**: 역할 선택 토글 또는 라디오 버튼
- **4개 탭**: 대시보드 / 평가지 / 문항 / 분석
- **요약카드 4개**: 풀이 오답수, 미해결 오답수, 풀이율, 극복율 (숫자 + 아이콘)
- **Chart.js 차트**: 과목별 오답수 막대그래프, 추이 라인그래프
- **취약영역 분석 리스트**: 성취기준별 오답수/풀이율 (테이블 또는 카드)
- **문항 리스트/테이블**: 평가명, 정답률, 오답/미해결 여부, 오답 상세 팝오버

### 디자인 톤
- 색상: 파란색 주(#2563eb, #3b82f6), 빨강 경고(#ef4444)
- 카드: 흰색 배경, border-radius 12px, 미세 박스섀도우
- 폰트: Pretendard, 제목 16px/600, 본문 14px/400
- 배경: #f5f7fa (페이지), #ffffff (카드)

## 추가/보완 항목

### 1. 기본 조회 조건 개선 (필수)
**기존**: 조회 기간 미설정 또는 전체 기간만 표시
**추가 필요**:
- 상단에 **조회 기간 선택** 드롭다운 추가:
  - "전체기간" (기본값)
  - "최근 3개월"
  - "최근 6개월"
  - "이번 학기"
  - "직접 입력" (시작일~종료일 캘린더)
- 선택 후 차트 및 리스트 자동 갱신
- 선택한 기간 우측에 작은 텍스트로 "(2024.09~2025.03)" 표시

### 2. 필터 고도화 (필수)
**기존**: 필터 미구현 또는 단순 교과 선택만 가능
**추가 필요**:
- **필터 영역**: 상단 또는 좌측 사이드바에 배치
- **필터 항목** (선택 가능한 것부터):
  - 과목: 여러 선택 (체크박스) - 국어, 영어, 수학, 사회, 과학 등
  - 단원/주제: 드롭다운 (과목 선택 후 활성화)
  - 성취기준: 드롭다운 (단원 선택 후 활성화, 예: "12~가2~01")
  - 해결여부: 라디오 버튼 (전체/미해결만/해결됨)
  - 중요도 (선택): 별점 이상 필터 (⭐⭐⭐ 이상 등)
- **필터 버튼**: "초기화" 버튼으로 모든 필터 리셋 가능
- 선택된 필터 항목 개수 배지 표시 (예: "필터 3개 적용")

### 3. 학생 주도 관리 (권장)
**추가 필요**:
- **태그 기반 필터링**: 각 오답 문항에 학생이 태그 추가 가능
  - 태그 입력 필드: 우측 "태그 추가" 링크 또는 아이콘
  - 태그 예시: #취약점 #실수 #개념부족 #계산실수 등
  - 태그 자동완성: 기존 태그 드롭다운
- **태그 필터 칩**: 상단에 "추가한 태그 칩" 표시, 클릭 시 해당 태그만 필터링
- **즐겨찾기/메모**: 문항별 "즐겨찾기" 별 아이콘 + "메모" 텍스트 에리어
  - 메모는 아이콘 클릭 시 모달 팝업으로 편집

### 4. 게이미피케이션 (선택)
**추가 필요**:
- **포인트 시스템**: 우측 상단에 "포인트 아이콘 + 누적점수" 표시
  - 오답 해결 시 +10 포인트, 문항 메모 시 +5 포인트 등
  - 클릭 시 포인트 획득 이력 모달 팝업
- **배지/배장**: 성취기준별로 달성하면 배지 표시
  - "국어 완벽한 이해 🏆", "수학 70% 이상 극복 🌟" 등
  - 배지 진열 섹션: 대시보드 하단 또는 별도 탭
- **연속해결 스트릭**: 일일 해결한 오답 수를 화염 아이콘(🔥)으로 표시
  - "🔥 5일 연속 해결 중!" 배너 표시 (상단 또는 우측)
  - 선택: 월별 달성 달성도 캘린더 (GitHub 잔디처럼 그리드)

## 디자인 참고

### 색상 정의
```
오답 상태:
- 미해결: #ef4444 (빨강)
- 해결됨: #10b981 (초록)
- 부분정답: #f59e0b (주황)

포인트/게이미피케이션:
- 포인트: #8b5cf6 (보라)
- 배지: #fbbf24 (금색)
- 스트릭: #ff6b6b (주황빨강)
```

### 컴포넌트 명세
- **조회기간 드롭다운**: 높이 36px, 폭 200px, 좌측정렬
- **필터 칩**: 높이 28px, padding 6px 12px, border-radius 16px, 배경 #e0e7ff
- **태그 칩**: 높이 24px, 배경 #f0fdf4, 색상 #10b981
- **포인트 표시**: 우측상단 고정, 배경 흰색 카드, padding 12px, border-radius 8px
- **배지**: 40x40px, 원형, 배경 금색, 텍스트 센터정렬, 폰트 12px/700

### 레이아웃 조정
- 상단 조회기간/필터: 좌측에서 우측으로 수평 배치, 간격 16px
- 문항 테이블: "태그" 컬럼 추가 (우측에서 3번째), "메모" 아이콘 추가 (우측끝)

## 기존 구현 현황
- 개설/참여 클래스 탭, 클래스 바로가기 카드 (클래스 목록 그리드)
- 일정 캘린더 (월 네비게이션 + 범례: 수업/과제/평가/설문)
- 필터바 (기간/클래스/상태 필터)
- 클래스 현황 섹션: 통계 카드 4종 (출석현황/수업이수율/과제제출률/평가진행률)
- 콘텐츠 섹션: 새로 올라온 글 / 인기글 / 설문 (각 3건 목록 + 전체보기 버튼)
- 활동 내역 섹션 (카드 기반, 교사/학생 뷰 분리, 상태 색상, 진행률 바)
- dacheum-style.css 외부 스타일시트 연동

## 추가/보완 항목

### 1. 통계 카드 (stats-grid) 내용 보강
- 현재: 제목만 표시
- **필요**: 각 카드에 수치 + 텍스트 + 작은 인라인 아이콘 추가
  - 출석현황: "85.5%" + "출석" + 인원 "(총 28명 중)"
  - 수업이수율: "12/15" + "완료" + 진행률 바(80%)
  - 과제제출률: "18/20" + "제출" + 진행률 바(90%)
  - 평가진행률: "4/6" + "진행중" + 진행률 바(66%)

### 2. 게시글 섹션 (content-sections) 확장
- 현재: 3건 목록만 표시
- **필요**:
  - 각 게시물 피드에 "작성자 아바타 + 작성자명 + 시간" 추가
  - 제목 아래 요약 텍스트(2줄) 또는 미리보기 추가
  - 좋아요/댓글수 아이콘 + 숫자 하단에 표시
  - "더보기" 또는 확장 인터랙션 추가 가능

### 3. 활동 카드 (activity-card) 공통 구성 요소 일관성
- 현재: 제목/상태/진행률 바만 표시
- **필요**:
  - 카드 상단: 과제/수업/평가 유형별 아이콘 + 타입 라벨
  - 카드 중앙: 제목 + 설명 텍스트 (1줄)
  - 카드 하단: 진행률 바 + 숫자 퍼센트
  - 우측: 상태 배지 (진행중/완료/예정/기한임박)
  - 호버 상태: 약간의 그림자 강화, 컬러 하이라이트

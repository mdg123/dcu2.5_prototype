## 추가/보완 항목

### 1. 인라인 스마트 프롬프트 (필수)
**기존**: 메타데이터 입력 필드 분산 (과목, 학년, 단원 등 드롭다운만)
**추가 필요**:
- **콘텐츠/과제 작성 시 "스마트 입력 모드" 추가**:
  - 제목 또는 설명 입력 중에 "스마트 프롬프트" 버튼 (⚡ 또는 💡 아이콘)
  - 버튼 클릭 시 **메타데이터 검색바** 팝오버 표시
  - 예: "함수의 극한 개념 강의" 입력 → "학교급/학년/교과/성취기준/난이도" 자동 제안

#### 스마트 프롬프트 동작 흐름
```
사용자 입력: "함수의 극한"
         ↓
검색 분석: "함수" → 수학과목
         ↓
자동 제안:
├─ 학교급: 고등학교
├─ 학년: 고2~3학년
├─ 교과: 수학
├─ 성취기준: [12수학I01-01] 함수의 극한
├─ 난이도: 상(심화)
└─ 단원: 함수와 극한 > 함수의 극한
         ↓
사용자 선택/수정 후 자동 입력
```

### 2. 실시간 자동완성 드롭다운 (필수)
**기존**: 드롭다운 클릭 후 목록 선택 (사용자 입력 미지원)
**추가 필요**:
- **각 메타데이터 필드에 "검색 자동완성" 추가**:
  - 학교급/학년/교과/성취기준/난이도 드롭다운에서
  - 사용자가 글자 입력 시 실시간 필터링
  - 예: "수" 입력 → 수학, 수사, 수리과학 등 검색결과 표시

```javascript
// 자동완성 예시
const autocompleteSuggestions = {
  '교과': ['국어', '영어', '수학', '사회', '과학', '체육', '음악', '미술'],
  '성취기준': [
    '[12수학I01-01] 함수의 극한',
    '[12수학I01-02] 함수의 연속성',
    // ...
  ]
};

// 입력 시 필터링
function filterSuggestions(input, category) {
  return autocompleteSuggestions[category].filter(item =>
    item.toLowerCase().includes(input.toLowerCase())
  );
}
```

### 3. 검색어 입력 시 드롭다운 영역별 배지 (필수)
**추가 필요**:
- **검색 결과 드롭다운에 "영역별 배지" 표시**:
  ```
  [🏫 학교급] 초등학교 / 중학교 / 고등학교
  [📚 학년] 1학년 / 2학년 / 3학년
  [📖 교과] 국어 / 수학 / 영어 / 사회 / 과학 ...
  [🎯 성취기준] [12수학I01-01] 함수의 극한
  [⭐ 난이도] 하(기초) / 중(표준) / 상(심화)
  ```
- 각 영역 헤더 위에 배지 표시 (배경색 구분)
- 배지 색상: 학교급(파랑) / 학년(초록) / 교과(주황) / 성취기준(보라) / 난이도(빨강)

### 4. 키보드 방향키+엔터키 탐색 접근성 (필수)
**추가 필요**:
- **드롭다운 키보드 네비게이션**:
  - 화살표 키(↑/↓): 드롭다운 항목 이동
  - 엔터키: 현재 하이라이트된 항목 선택
  - Escape: 드롭다운 닫기
  - Tab: 다음 필드로 이동

```javascript
// 키보드 이벤트 예시
document.querySelector('.autocomplete-input').addEventListener('keydown', (e) => {
  if (e.key === 'ArrowDown') {
    // 다음 항목으로 이동
  } else if (e.key === 'ArrowUp') {
    // 이전 항목으로 이동
  } else if (e.key === 'Enter') {
    // 현재 항목 선택
  } else if (e.key === 'Escape') {
    // 드롭다운 닫기
  }
});
```

### 5. 표준코드 자동변환 (권장)
**추가 필요**:
- **성취기준 입력 후 자동 코드 변환**:
  - 사용자가 "함수의 극한" 선택 → 자동으로 "[12수학I01-01]" 코드 입력
  - 선택된 성취기준 아래에 "코드: [12수학I01-01]" 회색 텍스트로 표시
- **학교급/학년 자동 입력**:
  - 고등학교 과목 선택 → "학년" 필드에 "고1~3" 자동 선입력
  - 중학교 과목 선택 → "학년" 필드에 "중1~3" 자동 선입력

### 6. 저장된 프리셋 (선택)
**추가 필요**:
- **자주 사용하는 메타데이터 조합을 "프리셋"으로 저장**:
  - 예: "수학 고2 함수 상" → 프리셋 저장 (⭐ 버튼)
  - 다음 작성 시 "프리셋 불러오기" → 자동 입력
  - 프리셋 관리: 추가/삭제/순서변경

## 메타데이터 표준 데이터셋

### 학교급
- 초등학교 (1~6학년)
- 중학교 (1~3학년)
- 고등학교 (1~3학년)

### 교과 (2022 개정)
- 국어, 영어, 수학, 사회, 과학
- 체육, 음악, 미술
- 정보, 기술·가정
- 교양, 한문, 논술

### 성취기준 예시 (수학)
- [12수학I01-01] 함수의 극한
- [12수학I01-02] 함수의 연속성
- [12수학I02-01] 미분의 정의
- [12수학I02-02] 미분의 활용
- (교과별 40~50개)

### 난이도
- 하 (기초학습)
- 중 (표준학습)
- 상 (심화학습)

### 단원 예시 (수학 고1)
- 함수와 극한
  - 함수의 개념
  - 함수의 극한
  - 함수의 연속성
- 미분
  - 미분의 정의
  - 미분의 활용
- 적분
  - 적분의 정의
  - 부정적분과 정적분

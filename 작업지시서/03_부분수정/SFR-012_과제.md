## 기존 구현 현황
- 교사/학생 역할 전환 토글
- Quill 에디터 (과제 설명/풀이용)
- 과제카드 리스트 (상태배지/제출률게이지바)
- 카드 확장 상세 보기
  - 교사: 제출현황 테이블, 채점 입력 패널
  - 학생: 에디터, 파일 업로드 D&D
- 표절률 배지, AI 표절 감지 배지
- 파일 업로드 드래그&드롭
- 채점 입력 자동저장 메커니즘

## 추가/보완 항목

### 1. 과목 탭 커스터마이징
- 현재: 고정된 과목 탭 (전체/수학/국어/영어)
- **필요**:
  - 각 탭 우측에 설정(⚙) 아이콘 표시
  - 클릭 시 모달 팝업: "탭 순서 변경" 또는 "탭 커스터마이징"
  - 모달 내용:
    - 드래그로 탭 순서 변경 가능
    - 체크박스로 탭 표시/숨김 토글
    - 새 과목 추가 입력창 (옵션)
  - 저장 후 탭 즉시 반영

### 2. 필터/정렬 강화
- 현재: 기본 정렬만 제공
- **필요**: 정렬 드롭다운에 옵션 추가
  - 마감임박순 (D-day 오름차순)
  - 제출률순 (낮은 순 → 높은 순)
  - 생성날짜순 (최신 우선)
  - 기한 내림차순 (먼 날짜 우선)

### 3. 상세 화면 아코디언 구조
- 현재: 전체 카드 펼침
- **필요**:
  - 카드 클릭 시 부분 펼침 (헤더 + 핵심 정보만)
  - 아코디언 섹션:
    - "과제 정보" (펼침 가능): 설명/파일/기한 등
    - "제출 현황" (교사용): 학생별 제출/미제출 테이블
    - "채점 현황" (교사용): 채점 입력 폼
    - "내 제출" (학생용): 업로드/에디터 영역
  - 펼침/접힘 애니메이션 (chevron 아이콘 회전)

### 4. 인라인 채점 자동저장
- 현재: 저장 버튼 클릭 필요
- **필요**:
  - 점수 입력 필드: onChange 이벤트로 즉시 저장
  - 피드백 textarea: 3초 디바운스 후 자동저장
  - 저장 상태 표시: "저장 중..." → "✓ 저장됨" (1초 유지 후 사라짐)
  - 저장 실패 시 경고 토스트

### 5. 피드백 WYSIWYG
- 현재: plain textarea
- **필요**:
  - Quill 에디터 또는 간단한 WYSIWYG 도구 적용
  - 기능: 굵게/기울임/밑줄, 색상, 목록, 링크
  - 문자수 제한 표시 (최대 2000자)
  - 미리보기 버튼 (학생 입장에서 보기)

### 6. AI 제출물 요약 Mock 카드
- **필요**: 학생 제출 내용 상단에 AI 요약 카드 추가
  - 제목: "AI 요약"
  - 내용: "이 제출물은 [주제]에 대해 [2~3줄 요약]..."
  - 아이콘: ✨ 또는 🤖
  - 스타일: 밝은 파란색 배경 (#E8EDFF), 테두리 (#4A6CF7)
  - 닫기 버튼: X 아이콘

## 디자인 참고
- **색상**: 기존 스타일 유지

## 기존 구현 현황

### 현재 보유 기능
- ✓ SPA 해시 라우팅: `#my-storage` 하위 탭 구조
- ✓ 나의 보관함 3차 탭: 내자료, 담은자료, 구독채널, 채널관리
- ✓ 채널관리 탭: 기본 채널 목록 표시
- ✓ 채널 카드: 채널명, 기본 정보

## 추가/보완 항목

### 1. 채널 페이지 분리 (독립 URL 구조)
**현재**: 채널관리 탭이 나의 보관함 내부 탭으로만 존재
**개선**:
- 채널명 클릭 시 채널 전용 페이지로 이동
- URL 해시: `#channel-detail/{channelId}` 또는 `#channel-view`
- 뒤로가기 버튼: "나의 보관함" 링크로 복귀
- 채널 페이지 GNB 내 경로 표시: "나의 보관함 > 채널관리 > [채널명]"
- 모바일에서 풀스크린 뷰

### 2. 채널 페이지 브랜드화 UI
**위치**: 채널 상세 페이지 상단

#### 2-1. 채널 배너 & 프로필
- **배너 영역**: 가로 1200px, 높이 240px (또는 반응형)
  - 기본 배경: 그라데이션 (예: #2563eb → #1d4ed8)
  - 배너 이미지 교체 버튼: "배너 변경" (우측 상단, 편집 권한 시만)
  - 파일 업로드: 2MB 제한, jpg/png 형식

- **채널 프로필 영역**: 배너 하단 (겹침)
  - 프로필 아이콘: 원형 120px × 120px (테두리 4px white)
  - 채널명: 18px, weight 700, #1f2937
  - 채널 설명: 13px, weight 400, #6B7280 (최대 2줄)
  - 프로필 이미지 교체 버튼: "이미지 변경" (마우스 호버 시 표시)

#### 2-2. 채널 통계 바
- 위치: 프로필 하단, 수평 구분선 상단
- 요소:
  - 폴로워 수: "1,234명 팔로우"
  - 콘텐츠 수: "456개 콘텐츠"
  - 누적 조회수: "5,678회"
  - 구독자 증감: "+45 (이번달)" (우측, 소형)
- 스타일: 회색 배경 #f9fafb, padding 12px 16px, 수평 분할선

#### 2-3. 채널 액션 버튼 바
- 위치: 프로필 하단, 통계 바 우측 정렬
- 버튼들:
  - "팔로우 취소" (또는 "팔로우") - Primary 블루
  - "공유" - Secondary 회색
  - "더보기" ⋯ - Ghost 투명 (메뉴: 신고, 차단 등)
- 스타일: 버튼 height 36px, padding 8px 16px

### 3. 크리에이터 커뮤니티 전용 게시판 UI
**위치**: 채널 페이지 중앙 (프로필 하단)
**요소**:
- 탭 구조: "콘텐츠" | "커뮤니티" | "통계" (수평 탭 바)

**커뮤니티 탭 내용**:
- 섹션: "채널 소식" (13px, weight 600)
  - 공지사항 핀 아이콘: 1~3개 고정 게시물
  - 각 게시물: 제목 + 작성일 + 반응 수 (좋아요/댓글)

- 섹션: "팔로워 커뮤니티 게시판" (13px, weight 600)
  - 게시물 입력창 ("어떤 내용을 공유하시겠어요?")
  - 게시물 목록: 카드형 (작성자 | 내용 | 좋아요 | 댓글 | 작성일)
  - 댓글 수 (12px, #6B7280)
  - 페이지네이션: 10개/페이지

**게시물 카드 스타일**:
- 배경: 흰색 #FFFFFF
- border-radius: 8px
- padding: 12px 16px
- 호버: 배경 #f9fafb, 그림자 강화

### 4. 고급 편집 도구
**위치**: 채널 프로필 영역 우측 (편집 권한이 있는 크리에이터만)
**요소**:
- 버튼: "채널 설정" (또는 "편집") - Secondary 회색
- 클릭 시 모달 열기:

**채널 설정 모달** (3탭):
1. **기본 정보**
   - 채널명: text input (100자 제한)
   - 채널 설명: textarea (500자 제한)
   - 범주: dropdown (과학, 영어, 수학, 역사 등)
   - 채널 공개 여부: toggle (공개/비공개)

2. **브랜딩**
   - 배너 이미지: 파일 업로드 (2MB)
   - 프로필 이미지: 파일 업로드 (2MB)
   - 채널 컬러 테마: 색상 선택기 (Primary 색상 커스터마이징)

3. **멤버 관리**
   - 채널 관리자: 이름 | 권한 | 제거
   - 관리자 초대: "이메일 추가" input + "초대" 버튼
   - 역할: 채널 관리자, 콘텐츠 검수자 (드롭다운)

**모달 스타일**:
- width: 600px (반응형)
- border-radius: 12px
- 하단: "저장" (Primary) / "취소" (Ghost) 버튼

## 주의사항

1. **SPA 네비게이션 유지**: 기존 `#my-storage` 해시 라우팅과 충돌 없이 새로운 `#channel-detail` 라우팅 구현
2. **편집 권한 검사**: "채널 설정" 버튼은 채널 소유자/관리자만 볼 수 있도록 로직 추가
3. **이미지 크기 제한**: 배너(2MB) 및 프로필(2MB) 파일 검증 필수
4. **게시판 데이터**: 더미 게시물 데이터는 팔로워 수, 조회수와 일관성 있게 구성
5. **성능**: 게시물이 많을 경우 무한스크롤 또는 페이지네이션으로 성능 최적화

## 기존 구현 현황
- 5개 화면 분리 (목록/생성/설정/결과대시보드/CBT풀이)
- 과목탭 + 필터 (기간/상태)
- 정오답 그리드 (sticky 헤더)
- 문항분석 바 차트
- 서술형 채점 패널
- 성취기준 분석
- CBT 사이드바 + 문항 풀이 UI

## 추가/보완 항목

### 1. 평가지 자동구성 모드
- 현재: 수동으로 문항 추가만 가능
- **필요**:
  - "AI 자동구성" 버튼 추가 (생성 화면 상단)
  - 클릭 시 모달:
    - 입력: 교육과정/성취기준 선택 드롭다운
    - 선택: 문항 수, 난이도, 문제 유형
    - "생성" 버튼 → Mock 평가지 자동 생성
  - 생성된 평가지: 문항 목록 자동 입력, 각 문항 수정 가능

### 2. 문항 검색 + 보관함 연동 UI
- 현재: 문항을 수동으로만 입력
- **필요**:
  - "보관함에서 문항 선택" 버튼 추가
  - 팝업/모달:
    - 검색창 (키워드/성취기준 검색)
    - 기존 문항 목록 (카드 또는 테이블)
    - 체크박스로 여러 문항 선택
    - "추가" 버튼 → 선택한 문항 평가지에 추가
  - 선택된 문항 개수 배지 표시

### 3. 주관식 미채점/채점완료 상태 강화
- 현재: 기본 채점 상태만 표시
- **필요**:
  - 정오답 그리드에서 미채점 문항을 시각적으로 강조
    - 미채점: 배경색 주황색(#FEF3C7) + "채점 필요" 라벨
    - 채점중: 배경색 노란색(#FBBF24)
    - 채점완료: 배경색 초록색(#D1FAE5) + "✓" 아이콘
  - 채점 패널에서 "다음 미채점" 버튼으로 빠르게 이동

### 4. Chart.js 실제 차트 렌더링
- 현재: 차트 placeholder
- **필요**:
  - 문항분석 바 차트: 각 문항별 정답률 (가로 바 차트)
  - 성취기준 분석: 기준별 평균 점수 (원형/도넛형 차트)
  - 학급 분포: 점수대별 학생 분포 (히스토그램)
  - 옵션: responsive, legend, tooltip 활성화
  - Mock 데이터로 구성 (20명 학생, 10문항 샘플)

## 디자인 참고
- **색상**:
  - 미채점: #FEF3C7 (배경), #F59E0B (라벨)
  - 채점중: #FBBF24
  - 채점완료: #D1FAE5 (배경), #10B981 (텍스트)
  - Chart 색상: 정답률순 그라데이션 (초록~노랑~빨강)
- **타이포**: 문항 제목 14px, 상태 라벨 12px
- **차트**: Canvas height 300px, margin 20px

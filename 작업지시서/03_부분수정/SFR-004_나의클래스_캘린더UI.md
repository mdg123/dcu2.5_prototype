## 기존 구현 현황
- 기본 캘린더 그리드 (7열 × 6행)
- 월 네비게이션 (이전/현재월/다음 버튼)
- 범례 아이콘 (수업/과제/평가/설문)
- 상세 캘린더 모달 (openDetailCalendar() 함수)
- 날짜 셀에 단일 일정 점(dot) 표시만 가능

## 추가/보완 항목

### 1. 요약/상세 캘린더 뷰 전환
- 기본값: 요약 보기 (현재 구현)
- **필요**: "상세 캘린더 보기" 버튼 클릭 시 전환
  - 요약 보기: 작은 크기, 점(dot)으로만 일정 표시
  - 상세 보기: 더 큰 크기, 각 날짜에 일정 제목/시간 텍스트 표시
  - 토글 버튼: "요약" / "상세" 선택 가능

### 2. 기간 중첩 표현 (Bar 형태)
- 현재: 점으로만 표시
- **필요**: 여러 날에 걸친 일정을 시각화
  - 예: 2월 10일~15일 "모둠 프로젝트" → 날짜 셀 위에 컬러 바(막대)로 표시
  - 시작일: 왼쪽 모서리 둥근, 중간/끝일: 매끄러운 막대 형태
  - 색상: 일정 유형별로 구분 (수업/과제/평가/설문 각각 다른 색)

### 3. 일정 유형별 색상 라벨
- 범례 아이콘 색상 적용
  - 수업: 파란색 (#3b82f6)
  - 과제: 주황색 (#f59e0b)
  - 평가: 빨간색 (#ef4444)
  - 설문: 보라색 (#8b5cf6)
- 각 막대/점에 해당 색상 적용

### 4. 날짜별 상태 마커
- 오늘 날짜: 원형 테두리 강조 또는 배경 색 칠하기
- 과거 날짜: 회색(disabled) 처리 가능
- 미래 일정: 일반 색상 유지

### 5. 주간/월간 뷰 전환
- 토글 버튼 추가: "주간" / "월간" 선택
- 주간 뷰: 선택된 주의 7일 × 일정 시간 표시 (가로 스크롤 시간표 형태)
- 월간 뷰: 현재 구현 유지

## 기존 구현 현황

### 현재 보유 기능
- **Canvas 학습맵**: 노드 렌더링, 연결선, 드래그 이동, 마우스휠 줌/팬
- **좌측 필터 사이드바**: 학교/학년/영역/상태/학기 필터 항목
- **뷰 토글**: 맵(Canvas) / 리스트(테이블) 전환
- **모드 토글**: 단원 / 차시 레벨 전환
- **노드 상세패널**: 선택 노드 정보, 태그, 설명, 관련콘텐츠
- **대시보드 탭 3종**: 랭킹(순위표) / 커뮤니티(의견공유) / 학습기록(진행도)
- **Excel 업로드**: XLSX 파일 파싱, 학습노드 추가/수정
- **상태범례**: 아이콘 및 색상 범례 표시

### 디자인 톤
- 색상: 파란색 주조 (#2563eb, #3b82f6), 흑백 보조
- Canvas 배경: 라이트 그레이 (#f5f7fa)
- 노드 스타일: 원형 또는 사각형, 배경색으로 상태 구분
- 폰트: Pretendard, 제목 14px/600, 본문 12px/400

## 추가/보완 항목

### 1. 노드 상태별 색상 코딩 강화 (필수)
**기존**: 단순 색상 구분
**추가 필요**:
- **완료 노드**: 초록색 계열 (#10b981) + 체크마크 (✓)
- **진행중 노드**: 파란색 계열 (#3b82f6) + 진행률 퍼센트 (%)
- **미시작 노드**: 회색 계열 (#d1d5db) + 잠금해제 가능 표시
- **잠김 노드**: 어두운 회색 (#6b7280) + 자물쇠 아이콘 (🔒)
- **노드 우측하단**: 상태 배지 또는 미니 진행바 (5px 높이)
- Canvas 렌더링 시 상태에 따른 border 굵기/스타일 차별화

### 2. Dual Option 진입 (필수)
**추가 필요**:
- 노드 클릭 시 상세패널에 **2가지 버튼 노출**:
  - 버튼 1: "🚀 바로 학습" (학습카드 목록으로 이동)
  - 버튼 2: "📝 진단하기" (진단평가 풀이 인터페이스로 이동)
- 버튼 스타일: 파란색 주(바로학습) / 주황색 보조(진단하기)
- 호버 효과 및 터치 영역 최소 44x44px

### 3. 진단결과 UI 3단계 (필수)
**추가 필요**:
- 진단 완료 후 결과 노드 표시:
  - **준비완료** (초록, ✓): 해당 단원 학습 가능
  - **보완필요** (주황, ⚠): 선행학습 필요
  - **기초필요** (빨강, ⚠⚠): 기초학습부터 시작 권장
- 각 단계별 아이콘/색상/텍스트 통일
- 결과 상세 패널: 진단점수, 권장학습경로 텍스트, "학습시작" 버튼

### 4. 학습경로 시각화 (권장)
**추가 필요**:
- **경로 하이라이트**: 선택한 노드에서 학습 권장 경로 강조
- Canvas에 권장경로 연결선을 굵고 밝은 색상(#fbbf24 또는 #3b82f6)으로 표시
- 경로 상의 노드들을 순차적으로 번호 표시 (1️⃣ → 2️⃣ → 3️⃣ ...)
- 경로 추적 버튼: "경로 따라가기" → 자동 줌/패닝 및 노드 순차 강조

### 5. 부가서비스 표시 (선택)
**추가 필요**:
- 노드 상세패널 하단에 **"더보기" 섹션** 추가:
  - 문제집 링크 (📚 수능기출 문제집 100문항)
  - 추가콘텐츠 링크 (🎥 관련 동영상 강의)
  - 아이콘 + 텍스트 + 화살표 (→)
- 클릭 시 채움콘텐츠 또는 CBT 페이지로 이동

### 색상 정의
```
상태별 색상:
- 완료: #10b981 (초록)
- 진행중: #3b82f6 (파랑)
- 미시작: #d1d5db (연회색)
- 잠김: #6b7280 (짙은회색)
- 준비완료: #10b981 (초록)
- 보완필요: #f59e0b (주황)
- 기초필요: #ef4444 (빨강)
- 경로하이라이트: #fbbf24 (노랑)
```

### 노드 사이즈 및 스타일
- 노드 기본 크기: 60x60px (원형) 또는 70x50px (사각형)
- 텍스트: 12px/600, 센터 정렬
- 상태 배지: 우측하단 20x20px, 배경색 + 아이콘/숫자
- Border: 2px (활성) / 1px (비활성), 색상은 상태별

### 패널 및 버튼
- 상세패널: 우측 350px, 배경 흰색, border-radius 8px
- 버튼: 높이 40px, 폰트 14px/600, 아이콘 좌측 정렬
- 경로추적 버튼: 사이드바 상단, 높이 36px, 보라색 (#8b5cf6)

## 기존 구현 현황

### lesson-board.html
- 통계 카드 4개 (총 수업/이수/진행중/종료)
- 상태탭 (전체/진행중/종료/예정), 정렬 드롭다운, 검색바
- 수업카드: 번호/제목/태그/상태/진행률 바, 페이지네이션

### dacheum-lesson-player (4).html
- 2단 레이아웃 (좌: 사이드바 320px, 우: 콘텐츠 영역)
- 수업목차 (섹션별 접기/펼치기, 상태별 색상 아이콘)
- 문항 UI (객관식/주관식/서술형 지원)
- 사이드바 축소 토글 버튼

## 추가/보완 항목

### 1. 통합 뷰어 3분할 레이아웃
- 현재: 좌측 사이드바 + 중앙 콘텐츠 (2분할)
- **필요**: 우측에 소통 패널 추가 (3분할)
  - 좌측 (240px): 수업목차 네비게이션
  - 중앙 (calc(100% - 240px - 320px)): 콘텐츠 뷰어
  - 우측 (320px): 소통/채팅 패널 (기본 숨김, 토글 가능)
  - 우측 패널 토글 버튼: 중앙 헤더에 "💬" 아이콘 버튼

### 2. 실시간 채팅 슬라이딩 패널 (우측)
- 우측 패널이 활성화되면 표시
  - 헤더: "실시간 채팅" 제목
  - 메시지 목록: 스크롤 가능, 송신자/수신자 메시지 구분 (좌측/우측 정렬)
  - 입력창: 텍스트 입력 + 전송 버튼
  - Mock 데이터: 교사/학생 메시지 샘플 3~4개

### 3. 판서도구 오버레이
- 콘텐츠 영역 상단 교사 전용 도구모음
  - 버튼: 펜/지우개/형광펜/모양그리기/실행취소/초기화
  - 색상 팔레트 (5개: 검은색/빨간색/파란색/초록색/노란색)
  - 펜 굵기 선택 슬라이더 (1px ~ 5px)
  - Canvas 기반 간단한 드로잉 기능 (실제 그리기는 선택사항)

### 4. 형성평가 팝업
- 학습 중간중간 미니 퀴즈 팝업 (1~3문항)
  - 모달: "빠른 확인" 제목
  - 문항: 객관식 4지선다
  - 버튼: "건너뛰기" / "제출"
  - 제출 후: 결과 메시지 ("정답입니다!" 또는 "틀렸습니다")

### 5. 개인화 설정 (사용자 환경설정)
- 헤더 우측 설정 아이콘 (⚙)
  - 다크모드 토글
  - 텍스트 크기 선택 (작음/보통/큼)
  - 단축키 안내 (팝업)
  - 설정값을 localStorage에 저장

### 6. 이전/다음 고정 네비게이션
- 콘텐츠 하단: "< 이전 문항" / "다음 문항 >" 버튼 (고정 위치, 항상 보임)
- 또는 사이드바에 화살표 버튼으로 구현
- 현재 문항 번호 표시

## 디자인 참고

  - 텍스트: #1F2937, Gray: #E5E7EB
- **레이아웃**:
  - 우측 패널: white background, 위쪽 고정 헤더 48px
  - 메시지: 송신 우측(파란색), 수신 좌측(회색)

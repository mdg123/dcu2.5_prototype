## 기존 구현 현황

### 클래스 대시보드 현황
- **클래스 헤더**: 학급명, 교사명, 학기 정보
- **핵심지표 4개** (CSS 도넛차트):
  - 감정출석률
  - 기초력진단 평균점수
  - 일일학습 목표달성률
  - 독서활동 권장량 달성률
- **관심필요학생 경고리스트**: 학생명 + 위험도 배지 (빨강/주황) + "피드백" 버튼
- **4개 탭**: 학습현황 / 평가결과 / 수업참여도 / 과제제출

### 학생 개별보고서 현황
- **학생 프로필 헤더**: 학년/반/번호/이름/사진
- **성장요약 카드**: 종합 평가 텍스트 + 강점/개선점 불릿리스트
- **6개 섹션 카드**: 오늘의학습 / 감정출석 / 기초력진단 / 학습분석 / 독서활동 / 선생님피드백

### 디자인 톤
- 색상: 파란색 주(#2563eb, #3b82f6), 초록 포인트(#10b981), 빨강 경고(#ef4444)
- 카드: 흰색 배경, border-radius 12px, 미세 박스섀도우
- 폰트: Pretendard, 제목 16px/600, 본문 14px/400
- 배경: #f5f7fa (페이지), #ffffff (카드)

## 추가/보완 항목

### 1. 대시보드 학생리스트 + 관심필요 플래깅 강화 (필수)
**기존**: 관심필요학생 리스트만 표시
**추가 필요**:
- **대시보드 상단에 "학생 현황" 섹션** 추가:
  - 전체 학생 수 + 현황 통계 (정상: 초록 / 주의: 주황 / 우려: 빨강)
  - 학생 리스트 테이블: 학번 / 이름 / 상태 아이콘 / 관심플래그 (별 또는 심장)
  - 상태 아이콘: 🟢 정상 / 🟡 주의 / 🔴 우려
- **관심필요학생 리스트 강화**:
  - 기존 "위험도 배지"에 추가로 "플래그 이유" 표시
  - 예: "학습부진 🚩" / "감정저하 🚩" / "출석불량 🚩"
  - 클릭 시 상세 분석 페이지로 이동
- **학생 행 우측에 "액션 버튼"**: "피드백" / "개인상담일정" / "학부모연락" 등

### 2. 성장 요약 문구 자동생성 영역 (권장)
**기존**: 수동으로 작성한 성장요약 텍스트
**추가 필요**:
- **학생개별보고서 상단**에 **"자동생성 요약" 섹션** 추가:
  - 학습분석 데이터(학습시간, 과제제출률, 감정, 참여도 등)를 기반으로
  - AI 또는 템플릿 기반 자동 문장 생성:
    ```
    "OOO 학생은 이번 학기 평균 일일 45분 학습으로 자기관리가 양호합니다.
    특히 과학 교과에서 수행평가 100% 제출하여 책임감이 돋보입니다.
    다만 수학 개념이해도가 75%로 보완이 필요합니다."
    ```
  - "📝 수정" 버튼: 자동생성 문구를 편집 가능한 입력필드로 변환
  - "🔄 재생성" 버튼: 다시 생성하여 다른 표현 확인

### 3. 선생님 관찰기록 (권장)
**추가 필요**:
- **학생개별보고서 하단에 "선생님 관찰기록" 섹션** 추가:
  - 자유형식 텍스트 에리어: "이 학생에 대한 관찰사항을 자유롭게 기록하세요"
  - "태그 추가": 관찰내용과 관련된 태그 추가 (#성실함 #리더십 #협력 등)
  - 태그 자동완성 드롭다운 (기존 태그 풀 활용)
  - "저장" 버튼: 관찰기록 저장 (서버 또는 localStorage)
  - 저장된 기록 타임라인 표시 (최근 3개 항목 표시)

### 4. 기간선택 (필수)
**기존**: 전체 학기만 표시
**추가 필요**:
- **대시보드 및 학생보고서 상단에 "기간 선택" 드롭다운** 추가:
  - "월별" (최근 3개월 선택 가능)
  - "주별" (최근 4주 선택 가능)
  - "학기별" (1학기 / 2학기)
  - "학년별" (전학년 / 현학년)
- 선택 후 모든 차트/통계 자동 갱신
- 선택한 기간 우측에 작은 텍스트로 표시 (예: "2025.01~2025.03")

### 5. 학부모 공개설정 토글 (권장)
**추가 필요**:
- **학생개별보고서 우측상단에 "학부모공개" 토글** 추가:
  - ON (기본): 학부모에게 공개됨 (🌐 공개)
  - OFF: 학부모 비공개 (🔒 비공개)
- 토글 변경 시 확인 메시지: "비공개로 변경하시겠습니까? 학부모는 이 보고서를 볼 수 없습니다."
- 공개상태 아이콘 표시 (상단 우측)

### 6. 내보내기 (선택/권장)
**추가 필요**:
- **학생개별보고서 우측상단 "내보내기" 버튼** 추가:
  - "PDF 다운로드" → 보고서 전체 PDF 다운로드
  - "인쇄" → 브라우저 인쇄 미리보기 열기
  - "학부모 공유 링크" → 공유가능한 고유 URL 생성 (학부모용 읽기전용)
- PDF에 포함: 학생정보 + 성장요약 + 6개 섹션 내용 + 교사의견

### 7. 사용자별 뷰 분리 (필수)
**추가 필요**:
- **GNB에 역할 선택** (또는 기존 로그인 역할 자동 감지):
  - 👨‍🏫 담임선생님 뷰 (기존)
  - 👨‍🎓 학생 뷰
  - 👩‍👩‍👧 학부모 뷰
- **담임선생님 뷰**:
  - 전체 클래스 대시보드 (학생리스트, 관심필요학생, 탭 분석)
  - 개별 학생 상세보고서 (편집가능, 관찰기록, 피드백)
- **학생 뷰**:
  - 개인 성장보고서만 읽기 (내 성장 요약)
  - "내 분석 데이터" 섹션 (오늘의학습, 감정, 진단, 독서 등)
  - 교사피드백 읽기만 가능
- **학부모 뷰**:
  - 공개된 자녀 성장보고서만 보기 (읽기전용)
  - "이번 학기 평가" 섹션 (분기별 성장 요약)
  - 문의하기 버튼 (교사에게 메시지 전송)

```
공개설정 아이콘:
- 공개: #3b82f6 (파랑)
- 비공개: #6b7280 (회색)
```

### 컴포넌트 명세
- **학생 현황 테이블**: 헤더 회색(#f3f4f6), 행높이 44px, 마우스오버 #f9fafb
- **관심필요 카드**: 배경 색상별 (🔴#fecaca / 🟡#fed7aa), 좌측 4px 경계선
- **자동생성 요약 박스**: 배경 #e0e7ff, 좌측 3px 파란색 경계선
- **기간선택 드롭다운**: 높이 36px, 폭 180px
- **학부모공개 토글**: 높이 24px, 우측상단 고정
- **관찰기록 타임라인**: 좌측 세로선 + 날짜 마커 + 텍스트 풍선

### 레이아웃 조정
- 대시보드: 상단에 "학생현황" 섹션 추가 (높이 300px)
- 학생보고서: 기간선택 + 학부모공개 + 내보내기 우측정렬 (상단)

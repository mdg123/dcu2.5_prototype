## 기존 구현 현황

- 일부 프로토타입에서 필터 상태 유지 구현 (CSS 클래스 토글)
- 사이드바 펼침/접힘 토글 존재 (일부 페이지)
- localStorage/sessionStorage 활용은 없음 (모두 세션 내 JS 변수)

---

## 추가/보완 항목

### 1. 검색 필터 설정 이력 관리 UI
- 검색/필터가 있는 페이지(클래스 찾기, 콘텐츠, CBT 등)에 **"최근 검색"** 영역 추가
- 위치: 검색바 하단 또는 사이드 패널
- 항목: 최근 사용한 필터 조합 (최대 5개), 각 항목에 요약 정보 표시
- 클릭 시 즉시 필터/설정 복원
- 개별 삭제(×) 버튼

### 2. 즐겨찾기 기능
- 자주 사용하는 필터 조합을 **"즐겨찾기"** 로 저장
- ⭐ 아이콘 클릭으로 현재 필터 상태 저장
- 저장된 즐겨찾기 목록 드롭다운

### 3. 설정 초기화 확인 팝업
- "초기화" 버튼 클릭 시 확인 모달:
  - "모든 필터 설정을 초기화하시겠습니까?"
  - [취소] [초기화] 버튼

### 4. 사이드바/위젯 상태 유지
- 사이드바 펼침/접힘 상태를 localStorage로 유지 (Mock: JS 변수로 시뮬레이션)
- 위젯 배치 순서 및 표시 여부 유지
- 대시보드 레이아웃 커스터마이징 상태 유지

### 5. 제작 이력 관리 UI
- 수업/과제/평가 제작 시 **"최근 설정"** 영역
- 이전에 사용한 설정 조합 표시 (최대 3개)
- 클릭 시 설정값 자동 채움

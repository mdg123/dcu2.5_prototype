## 기존 구현 현황
- 교사/학생 뷰 분리
- Sticky 테이블 (좌: 학생명+라벨, 우: 출석률)
- 기간 선택 (1주/2주/1달)
- 통계 카드 4개 (총 학생/출석자/지각/결석)

## 추가/보완 항목

### 1. 1클릭 출석 모드 (감정선택 없이)
- 현재: 체크인 → 감정 선택 필수
- **필요**:
  - "빠른 출석" 토글 옵션 (설정 패널)
  - 토글 ON: 체크인 버튼 클릭만으로 즉시 출석 확정 (감정 생략)
  - 토글 OFF: 기존 방식 유지 (감정 선택)
  - 토스트 피드백: "✓ 출석이 확정되었습니다"

### 2. 한마디 코멘트 입력란 (30자)
- **필요**:
  - 체크인 화면에서 감정 선택 아래에 텍스트 입력란 추가
  - 플레이스홀더: "오늘은 어땠나요? (30자 이내)"
  - 글자 수 표시: "5/30"
  - 선택사항: 코멘트 입력 후 제출
  - 제출된 코멘트: 교사용 리포트에 표시 가능

### 3. 엑셀 다운로드 버튼
- **필요**:
  - 교사 뷰 상단에 "📥 엑셀 다운로드" 버튼 추가
  - 클릭 시:
    - 선택된 기간의 출석 데이터 (.xlsx)
    - 컬럼: 학번, 이름, 날짜별 출석 상태 (O/X/지각/결석)
    - 하단: 통계 합계 행
  - 파일명: "출석부_2025-03-04.xlsx"

### 4. 출석 통계 바 그래프 (Chart.js)
- 현재: 통계 카드는 숫자만 표시
- **필요**:
  - 각 통계 카드에 가로 막대 그래프 추가
    - 출석: 녹색(#10B981), 지각: 주황색(#F59E0B), 결석: 빨강색(#EF4444)
    - 막대 높이: 전체 대비 비율
    - 라벨: 숫자 + 퍼센트 표시
  - Chart.js 사용 (horizontal bar chart)

### 5. 공동 목표 프로그레스 바
- **필요**:
  - 통계 카드 아래에 추가: "우리반 출석률 목표"
  - 프로그레스 바: 현재 85% → 목표 95%
  - 텍스트: "현재 85% / 목표 95% (까지 10% 남음)"
  - 색상: 목표 달성 시 초록색, 미달 시 주황색
  - 모바일: 스택 레이아웃

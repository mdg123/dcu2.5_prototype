## 추가/보완 항목

### 1. 계층형 폴더 관리 (3단계 트리뷰)
**위치**: 내자료 탭 좌측 사이드바 (또는 상단 네비게이션)
**구조**:
```
📁 내 자료실
  ├─ 📂 공개 (45개)
  │   ├─ 📂 2024 한국사 (23개)
  │   │   ├─ 1단원 (5개)
  │   │   ├─ 2단원 (8개)
  │   │   └─ 3단원 (10개)
  │   └─ 📂 2024 영어 (22개)
  ├─ 📂 비공개 (32개)
  │   ├─ 📂 초안 (15개)
  │   └─ 📂 검수 대기 (17개)
  └─ 📂 임시보관함 (8개)
```

**트리뷰 컴포넌트**:
- 각 폴더 좌측에 토글 화살표 (▶ / ▼)
- 폴더명 클릭 시 해당 폴더 콘텐츠 표시
- 폴더 개수 표시 (우측, 12px, #6B7280) - "(45개)"
- 우클릭 메뉴: 폴더 추가, 이름 변경, 삭제, 권한 변경

**스타일**:
- 배경: #f9fafb
- 폴더 아이콘: Font Awesome `fa-folder` (색상: #f59e0b)
- 선택된 폴더: 배경 #e0e7ff, 텍스트 #2563eb
- padding: 8px 12px, border-radius 6px

### 2. 드래그앤드롭 파일 이동
**기능**:
- 콘텐츠 카드를 좌측 폴더에 드래그 → 폴더로 이동
- 폴더 간 드래그 가능 (계층 구조 변경)

**UI 인디케이터**:
- 드래그 시 카드 투명도 0.5 + 커서 `grabbing`
- 드롭 가능한 폴더에 호버 시: 배경색 #dbeafe (라이트 블루)
- 드롭 완료 시 토스트: "콘텐츠가 [폴더명]으로 이동되었습니다."

**제약사항**:
- 비공개 폴더 → 공개 폴더 이동은 경고 모달: "공개 폴더로 이동하면 모든 사용자가 볼 수 있습니다. 계속하시겠습니까?"
- 동일 폴더 드래그는 무시

### 3. 유형별 필터 전환
**위치**: 콘텐츠 목록 상단 (검색창 옆)
**요소**:
- 필터 칩 (pill 형식): "전체", "수업", "과제", "평가", "자료"
- 선택된 필터: 배경 #2563eb, 텍스트 white
- 미선택: 배경 #f3f4f6, 텍스트 #6B7280
- 아이콘: 각 유형별 작은 아이콘
  - 수업: 📚
  - 과제: ✏️
  - 평가: 📋
  - 자료: 📄

**동작**:
- 클릭 시 해당 유형의 콘텐츠만 필터링
- 다중 선택 가능 (Ctrl/Cmd 클릭)
- "전체" 선택 시 모든 콘텐츠 표시

### 4. 보기 모드 전환 (카드형/목록형)
**위치**: 콘텐츠 목록 우측 상단
**요소**:
- 버튼 2개: "🔲 카드형" | "☰ 목록형"
- 선택된 모드: 배경 #2563eb, 텍스트 white
- 미선택: 배경 #f3f4f6, 텍스트 #6B7280

**카드형 뷰**:
- 그리드 레이아웃: 반응형 3~4열
- 각 카드: 썸네일 + 제목 + 메타정보

**목록형 뷰**:
- 테이블 구조: 체크박스 | 썸네일(작음) | 제목 | 유형 | 수정일 | 크기 | 상태
- 행 호버: 배경 #f9fafb

### 5. 다차원 태깅 시스템
**위치**: 콘텐츠 카드 하단 (또는 상세정보 영역)
**요소**:
- 각 콘텐츠에 여러 태그 가능
  - 학년 태그: #초등학교, #중학교, #고등학교
  - 과목 태그: #수학, #영어, #한국사
  - 커스텀 태그: #자유형식
- 태그 외형: pill 형식, padding 4px 8px, border-radius 4px
- 태그 색상: 배경 #f3f4f6, 텍스트 #1f2937
- X 버튼으로 태그 제거 가능

**태그 관리 UI**:
- 콘텐츠 카드 우측: "태그 추가" 버튼
- 클릭 시 모달: 기존 태그 검색 + 새 태그 입력
- 자동완성: 사용자가 이미 생성한 태그 목록 제시

**동작**:
- 태그 클릭 시 해당 태그 콘텐츠로 필터링
- 다중 태그 필터 가능 (OR 연산)

### 6. 콘텐츠 상태 5단계 세분화 라벨
**현재 상태**: 기본적인 공개/비공개만 구분
**확장**:
- 5가지 상태 라벨 배지:
  1. **임시저장** (회색 #6B7280): 아직 완성되지 않은 상태
  2. **검수대기** (주황색 #f59e0b): 검수자 승인 대기 중
  3. **반려됨** (빨강색 #ef4444): 검수 탈락 (반려 사유 팝오버)
  4. **공개** (초록색 #10b981): 모든 사용자에게 공개됨
  5. **비공개** (파랑색 #2563eb): 나만 볼 수 있음

**배지 표시**:
- 위치: 콘텐츠 카드 우측 상단 (또는 목록형에서 "상태" 열)
- 스타일: padding 4px 8px, border-radius 4px, font-size 11px, font-weight 600
- 호버: 상태 설명 팝오버 표시
  - 임시저장: "아직 작업 중입니다"
  - 검수대기: "검수자의 검토를 기다리고 있습니다"
  - 반려됨: "반려 사유: [메시지]" (+ "다시 제출" 버튼)
  - 공개: "모든 사용자가 이 콘텐츠를 볼 수 있습니다"
  - 비공개: "나만 볼 수 있습니다"

**상태 전환**:
- 임시저장 → 검수대기: "검수 신청" 버튼 (콘텐츠 상세 또는 우클릭 메뉴)
- 검수대기 / 반려됨 → 임시저장: "임시저장으로 돌아가기" 버튼
- 공개 ↔ 비공개: 토글 또는 "상태 변경" 메뉴

## 주의사항

1. **SPA 네비게이션 유지**: `#my-storage` 해시 라우팅 변경 금지
2. **기존 탭 구조 유지**: 내자료/담은자료/구독채널/채널관리 탭 순서 유지
3. **드래그앤드롭**: IE 11 미지원 (최신 브라우저만 대응)
4. **성능**: 콘텐츠가 1000개 이상일 경우 무한스크롤 또는 페이지네이션 고려
5. **더미 데이터**: 폴더 구조, 태그, 상태 데이터가 논리적으로 일관성 있어야 함
6. **권한 검사**: "상태 변경" 기능은 콘텐츠 소유자만 가능하도록 제약

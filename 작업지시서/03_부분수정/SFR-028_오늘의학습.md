## 기존 구현 현황

### 현재 보유 기능
- **학습설정**: 과목/학년/학기/난이도 드롭다운 4종
- **SVG 원형 진행률**: 76% 원형 게이지 (파란색)
- **목표 탭 3종**: 학습시간 / 과제제출 / 콘텐츠이용
- **7일 날짜 그리드**: 월-일 날짜 표시, 상태 아이콘 (완료✓ / 진행중◐ / 미시작○)
- **캐릭터**: 다채움이 이미지, Lv.12, 경험치 진행바
- **학습카드 그리드**: 8개 카드, 상태배지(★미완료 등), 태그(필수/선택), 진행률바

### 디자인 톤
- 색상: 파란색 계열 (#2563eb, #3b82f6)
- 카드: 흰색 배경, 보더레이디우스 12px, 박스섀도우 미세
- 폰트: Pretendard, 제목 16px/600, 본문 14px/400
- 레이아웃: 좌측 학습진행(320px) + 우측 학습카드 그리드(반응형)

---

## 추가/보완 항목

### 1. 교과/비교과 통합 구조 (필수)
**기존**: 과목 드롭다운에만 값 표시
**추가 필요**:
- 학습영역 토글 또는 탭: "전체 / 교과 / 비교과" (좌측 상단)
- 비교과 영역: 독서활동, 창체, 동아리 등 별도 학습카드 섹션
- 비교과 학습카드: 교과와 동일한 진행률/태그/상태배지 구조

### 2. 학습요소별 이수 추적 (필수)
**추가 필요**:
- 각 학습카드에 세부 이수요소 표시 (체크박스 형식 또는 진행도)
  ```
  [영상학습] 80% ✓
  [문항풀이] 진행중 5/10
  [문서열람] 완료 ✓
  ```
- 각 요소별 진행률 미니 게이지 (기존 카드 내부 추가)
- 호버시 상세 팝오버: 학습시간, 시작일시, 마지막학습 등

### 3. 주/월 결과표 UI (권장)
**추가 필요**:
- 기존 "7일 그리드" 우측에 "주별결과" 탭 추가
- 주단위 테이블: 주차 / 학습일수 / 목표달성도 / 누적시간
- 월단위 차트 (Chart.js): 주별 학습량 막대그래프

### 4. 나만의 문제집 기능 (권장)
**추가 필요**:
- 좌측 학습설정 하단에 "나만의 문제집" 버튼 추가
- 토글 또는 섹션 열림: 문제집 리스트 (이름/문제수/난이도/진행도)
- 각 문제집 카드에 "풀기" 버튼 → 모달 또는 새 페이지 (학습카드와 연계)

### 5. 포인트/랭킹 확장 (선택)
**추가 필요**:
- 캐릭터 정보 하단에 "포인트" 표시 (누적점수 + 일일획득)
- 미니 랭킹 카드: "오늘 반 랭킹 TOP 5" 표시
- 랭킹 진입 버튼 → 모달 또는 전체 랭킹 페이지 링크

## 디자인 참고

### 색상 추가
- 비교과: 보라색 계열 (#8b5cf6, #a78bfa)
- 상태배지: 초록(완료) / 주황(진행중) / 회색(미시작)

### 컴포넌트 명세
- **세부이수요소 체크**: 12px 체크박스 + 14px 텍스트 + 미니바 (8px 높이)
- **주별결과 테이블**: 헤더 회색(#f3f4f6), 행높이 40px, 마우스오버 #f9fafb
- **문제집 카드**: 90x100px 썸네일 + 텍스트, 진행바 상단, "풀기" 버튼 우측하단

### 레이아웃 조정
- 좌측 사이드바: 320px → 360px (추가공간)
- 캐릭터 섹션 높이: 220px → 280px (문제집 + 포인트 영역)

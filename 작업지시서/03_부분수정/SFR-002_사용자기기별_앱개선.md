# SFR-002: 사용자 기기별 앱 반응형 최적화

## 기본정보

| 항목 | 내용 |
|------|------|
| **대상파일** | 전체 프로토타입 (02~06 모든 폴더) |
| **현재상태** | 768px / 1024px / 1200px 브레이크포인트 일부 대응 |
| **수정유형** | 전체 반응형 개선 (크로스디바이스 최적화) |

| **예상작업시간** | 10-14시간 |

## 기존 구현 현황

### 현재 보유 기능
- **3단계 브레이크포인트**: 768px(태블릿) / 1024px / 1200px
- **일부 미디어쿼리**: 각 파일의 `<style>` 블록에 산재
- **반응형 그리드**: CSS Grid 또는 Flexbox 활용 (일부)
- **모바일 메뉴**: 해버거 메뉴 (일부 파일)

### 디자인 톤
- 폰트: Pretendard, 최소 14px (PC) → 12px (모바일)
- 간격: PC 24px 패딩 → 태블릿 16px → 모바일 12px
- 컬러: 기존 색상체계 유지

## 추가/보완 항목

### 1. PC 1920x1080+ 최적화 (필수)
**기존**: 최대폭 1200px 또는 1440px 고정
**추가 필요**:
- **전체 페이지 최대폭**: 1440px 이상 지원 (현재 고정 1200px → 1440px 확대)
- **타이포그래피 확대**:
  - PC 1920x1080 이상: 제목 20px~28px / 본문 15px~16px
  - 기존 PC 1200px: 제목 16px~24px / 본문 14px
- **사이드바 레이아웃**:
  - 사이드바 폭 280px → 320px (1920x1080 이상)
  - 콘텐츠 여백 24px → 32px 확대
- **카드 간격**: 16px → 20px 확대
- **테이블 행높이**: 40px → 48px 확대 (터치 안정성)
- **차트 크기**: Chart.js 캔버스 동적 리사이징 (최대 800x400px)

### 2. 태블릿 터치 최적화 (필수)
**기존**: 터치 영역 미지정 또는 28px 미만
**추가 필요**:
- **최소 터치 영역**: 44x44px (Apple HIG 기준)
  - 모든 버튼, 아이콘, 드롭다운 : 최소 44px
  - 현재 작은 버튼 (36px) → 44px 확대
  - 모바일 360px ~ 1024px에서 모든 대화형 요소 44x44px 이상 보장
- **버튼 간격**: 터치 요소 간 최소 8px 여유
- **입력필드**:
  - 높이: 44px (기본)
  - 폰트 크기: 최소 16px (iOS 오토줌 방지)
  - 터치 패딩: 상하 12px
- **드롭다운/셀렉트**: 높이 44px 이상, 옵션 텍스트 14px 이상
- **체크박스/라디오**: 24x24px (기본 16px → 확대)
- **슬라이더**: 높이 8px 트랙 + 32x32px 썸 (기본 4px + 16px)

### 3. 모바일 360x640~428x926 완전 대응 (필수)
**기존**: 모바일 레이아웃 부분적 대응
**추가 필요**:
- **초소형 모바일** (360px):
  - 카드 패딩: 16px → 12px
  - 폰트 크기: 14px → 13px (제목 15px / 본문 12px)
  - 메뉴 아이템 높이: 48px 유지
  - 글자짤림 방지 (max-width 100% 적용)
  - 테이블 가로스크롤 활성화 (min-width: 100vw)
- **소형 모바일** (375px):
  - 카드 패딩: 16px 유지
  - 폰트: 기본 14px
  - 2열 그리드 → 1열 그리드 강제
  - 사이드바 숨기기 (모바일 메뉴로 통합)
- **중형 모바일** (428px):
  - 가능하면 2열 그리드 허용 (대시보드 카드 등)
  - 사이드바 모바일 패널 (전체폭, 좌측 슬라이드인)
  - 폰트: 14px~15px
- **태블릿** (768px~1024px):
  - 2~3열 그리드 정상화
  - 사이드바 좌측 고정 또는 토글
  - 터치 최적화 44x44px 유지
  - 레이아웃: 가로 여백 16px~24px

### 4. 로그인 유지 UI (권장)
**추가 필요**:
- **로그인 화면 또는 계정 설정**에 "자동로그인" 토글 추가:
  - 토글 ON: 다음 방문 시 자동 로그인
  - 토글 OFF: 매번 로그인 필요
  - 공용 기기는 OFF 권장 안내문
- **프라이빗 모드/시크릿 모드**: "자동로그인" 자동 OFF 경고
- **만료시간 설정**: "7일 / 30일 / 항상" 옵션 (선택사항)

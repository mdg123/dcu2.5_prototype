## 기존 구현 현황
- 5가지 테마 (칠판/코르크/화이트보드/파스텔/클래식)
- contenteditable DIV 기반 에디터
- 검색 + 필터 (전체/공지/질문/공개)
- 댓글 + 반응 (좋아요/하트 등)
- 북마크 기능
- 초안 관리 (임시저장)
- 자동저장 배지

## 추가/보완 항목

### 1. WYSIWYG 에디터 통일 (Quill 적용)
- 현재: contenteditable DIV
- **필요**:
  - Quill 에디터로 통합
  - 도구모음: 굵게/기울임/밑줄, 색상, 제목/단락, 목록, 링크
  - 포맷 유지: 각 테마에 맞게 스타일링
  - 문자수 제한 표시 (최대 3000자)

### 2. 날짜 자동입력
- 현재: 수동 입력
- **필요**:
  - 작성/수정 시 현재 날짜/시간 자동 입력
  - 형식: "2025년 3월 4일 10:30" 또는 "3월 4일"
  - UI: 에디터 상단 또는 하단에 날짜 표시 (수정 불가)
  - 수정 시: "수정됨" 텍스트 추가

### 3. 미디어 삽입 개선
- 현재: 기본 파일 업로드만
- **필요**:
  - 클립보드에서 이미지 붙여넣기 (Ctrl+V)
  - 이미지 미리보기 (썸네일 표시)
  - URL 자동인식: "https://..." → 링크 자동 생성
  - 이미지/비디오 삽입 버튼 추가
  - 파일 크기 제한 표시 ("최대 5MB")

### 4. 임시저장 기능 강화
- 현재: 초안 저장만 가능
- **필요**:
  - 자동 임시저장: 5초 디바운스로 자동 저장
  - 저장 상태 UI: "저장 중..." → "✓ 임시저장됨"
  - 임시저장 목록: "임시저장 로드" 버튼 → 팝업에서 목록 선택
  - 로컬스토리지 활용 (5개까지 유지)

### 5. 인라인 뷰어
- 현재: 전체 페이지 뷰만 가능
- **필요**:
  - 알림장 목록에서 각 항목 클릭 시 미리보기 패널 (우측)
  - 슬라이드 아웃 패널: 너비 400px, 반투명 오버레이
  - 패널 내용: 제목/날짜/댓글/반응
  - 닫기 버튼: X 또는 Esc 키

### 6. 전체화면 모드
- **필요**:
  - 작성/보기 화면에서 전체화면 버튼 추가 (⛶ 아이콘)
  - 클릭 시 헤더/사이드바 숨김, 에디터 최대화
  - Esc 키로 종료

### 7. 읽음확인 / 재알림
- **필요**:
  - 교사 입장: 각 알림장 우측에 "읽음현황" 표시
    - "3/28명 읽음" 또는 진행률 바(10%)
  - 클릭 시 모달: 읽은 학생/안 읽은 학생 분리 표시
  - "미읽은 학생에게 재알림" 버튼 추가 (토스트 "재알림 발송됨")

### 8. 미리보기
- **필요**:
  - 발행 전 "미리보기" 버튼
  - 팝업: 학생 입장에서 보기 (댓글/반응 불가)
  - 테마 적용 확인 가능

## 디자인 참고
- **색상**: 기존 테마별 색상 유지
  - 칠판: 검은색/초록색, 코르크: 갈색, 화이트보드: 흰색/검은색
  - 텍스트 상태: #6B7280 (회색), 저장: #10B981 (초록)
- **타이포**: Pretendard, 본문 14px, 날짜 12px
- **Quill 높이**: 300px (기본), 500px (전체화면)

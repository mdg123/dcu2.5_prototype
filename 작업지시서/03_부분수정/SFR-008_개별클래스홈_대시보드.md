## 기존 구현 현황

- 모듈 On/Off 토글 (3개 모듈: 통계/활동/성취)

- 토스트 알림 구현

## 추가/보완 항목

### 1. 위젯 배치 순서 커스터마이징 (드래그)
- 현재: 고정된 위젯 배치
- **필요**:
  - 각 위젯 우측 상단에 "⋮" (3점 메뉴) 또는 "↕" (드래그 핸들) 아이콘 표시
  - 드래그 앤 드롭으로 위젯 순서 변경 가능하게 구현
  - 변경된 순서를 localStorage에 저장하여 새로고침 시에도 유지
  - 리셋 버튼: "기본 배치로 복원" 옵션 제공
  - 드래그 중: 위젯에 시각적 피드백 (그림자 강화, 배경 하이라이트)

### 2. 빈 상태(Empty State) UI
- 현재: 데이터 없을 때 처리 미흡
- **필요**:
  - 알림카드: 새로운 항목이 없으면 "최근 활동이 없습니다" 메시지 + 아이콘
  - 위젯: 데이터 없을 때 "데이터가 없습니다" 메시지 + 관련 액션 버튼 제공
  - 예: 최근활동 위젯 비어있으면 → "지금 과제를 만들어보세요" + "과제 생성" 버튼
  - 스타일: 회색 텍스트, 정중앙 정렬, 약한 배경색

### 3. Chart.js 실제 차트 렌더링 (현재 미구현)
- 현재: 차트 placeholder만 표시
- **필요**:
  - 학생성취 위젯: 성취기준별 도달율 바 차트 (horizontal bar chart)
  - 활동통계 위젯: 주간 활동 분포 (pie/doughnut chart)
  - 데이터: Mock 데이터로 구성 (실제 DB 연동은 나중)
  - 차트 옵션: responsive: true, maintainAspectRatio: true 설정
  - 범례/라벨 표시, 툴팁(hover) 기능 포함

## 디자인 참고
- **색상**: 기존 CSS 변수 활용
  - Primary Green: #4CAF50 (#primary-500)
  - Secondary Blue: #2196F3 (#secondary-500)
  - 배경: #FAFAFA (#gray-50)
  - 텍스트: #212121 (#gray-900)
- **드래그 UI**:
  - 핸들 아이콘: 회색 (#757575), 16px
  - 드래그 중 그림자: --shadow-lg 사용
  - 드롭 가능 위치: 밝은 파란색 테두리 (#90CAF9)
- **Empty State**: 아이콘 48px, 텍스트 14px, 회색 (#9E9E9E)

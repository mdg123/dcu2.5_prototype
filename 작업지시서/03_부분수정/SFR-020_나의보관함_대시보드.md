## 기존 구현 현황

### 현재 보유 기능
- ✓ SPA 해시 라우팅: `#my-storage` 통해 나의 보관함 페이지 표시
- ✓ 3차 탭: 내자료, 담은자료, 구독채널, 채널관리 (수평 탭바)

- ✓ 폴더 관리: 폴더 아이콘 표시, 기본 폴더 목록
- ✓ GNB 2차 메뉴 연동

## 추가/보완 항목

### 1. 채널 분석 요약 카드 추가
**위치**: 나의 보관함 대시보드 상단 (3차 탭 바 위)
**레이아웃**: 4개 카드 가로 배열 (또는 반응형 2행 2열)

**카드 내용**:
1. **내 채널 폴로워 수**
   - 아이콘: 📊 (또는 Font Awesome 유저 아이콘)
   - 수치: 1,234명 (18px, weight 700, #2563eb)
   - 레이블: "명" (12px, #6B7280)
   - 변화율: "+45 이번 달" (11px, #10b981, ↑ 아이콘)

2. **콘텐츠 총 개수**
   - 아이콘: 📁
   - 수치: 456개
   - 레이블: "개"
   - 변화율: "+12 이번 달"

3. **누적 조회수**
   - 아이콘: 👁️
   - 수치: 5,678회
   - 레이블: "회"
   - 변화율: "+234 이번 달"

4. **누적 좋아요 수**
   - 아이콘: ❤️
   - 수치: 890개
   - 레이블: "개"
   - 변화율: "+67 이번 달"

**스타일**:
- 카드 배경: 흰색 (#FFFFFF)
- 카드 테두리: 1px solid #e5e7eb
- border-radius: 8px
- padding: 16px
- 호버 효과: box-shadow 강화, 배경 #f9fafb로 미세하게 변경

### 2. 시계열 추이 라인 차트 추가
**위치**: 채널 분석 요약 카드 하단, 3차 탭 상단
**요소**:
- 차트 제목: "나의 채널 성과 추이" (14px, weight 600, #1f2937)
- 기간 선택: Pill 탭 (30일, 90일, 1년)
- Chart.js Line 차트
  - X축: 날짜 (일자 또는 주차)
  - Y축: 조회수/좋아요 수/폴로워 수 (선택 가능)
  - 범례: 3개 선 (조회수-파란색, 좋아요-초록색, 폴로워-주황색)
- 차트 높이: 250px
- 반응형: 모바일에서는 세로 스크롤 가능하도록

### 3. 즉시 행동 영역 (Action Cards)
**위치**: 시계열 추이 차트 하단, 3차 탭 상단
**용도**: 긴급 알림 또는 대기 중인 작업 표시

**카드 유형** (최대 3개까지 표시):
1. **반려된 콘텐츠 알림** (표시 조건: 반려 상태 콘텐츠 존재 시)
   - 배경: #FEE2E2 (라이트 레드)
   - 아이콘: ⚠️ (Danger 레드 #ef4444)
   - 텍스트: "반려된 콘텐츠 2개" (weight 600, #ef4444)
   - 버튼: "확인하기" (우측)

2. **검토 대기 콘텐츠 알림** (표시 조건: 검토대기 상태 콘텐츠 존재 시)
   - 배경: #FEF3C7 (라이트 오렌지)
   - 아이콘: ⏱️
   - 텍스트: "검토 대기 중인 콘텐츠 5개" (weight 600, #f59e0b)
   - 버튼: "확인하기"

3. **콘텐츠 공유 제안**
   - 배경: #DBEAFE (라이트 블루)
   - 아이콘: 💡
   - 텍스트: "비공개 콘텐츠 3개를 공개로 전환하시겠어요?" (weight 600, #2563eb)
   - 버튼: "지금 하기" 또는 "나중에"

**스타일**:
- border-radius: 8px
- padding: 12px 16px
- 마진: 8px 0
- 닫기 버튼: 우측에 X 아이콘 (cursor: pointer)

### 4. 최근 콘텐츠 성과 섹션
**위치**: 3차 탭 내용 상단 (내자료 탭 기본 표시)
**요소**:
- 섹션 제목: "최근 7일 인기 콘텐츠" (13px, weight 600, #1f2937)
- 보기 모드 전환 버튼: "카드형 | 목록형" (우측 정렬)
  - 선택된 모드는 배경 #2563eb, 텍스트 white
  - 미선택은 배경 #f3f4f6, 텍스트 #6B7280

**카드형 뷰**:
- 3~5개 콘텐츠 카드 가로 배열 (수평 스크롤 가능)
- 각 카드: 썸네일 + 제목(13px) + 조회수(12px) + 좋아요 수(12px)
- 호버 효과: 그림자 강화, 스케일 1.02

**목록형 뷰**:
- 테이블 형식: 순위 | 제목 | 조회수 | 좋아요 | 댓글 | 조회일시
- 헤더 배경: #f3f4f6
- 행 호버 배경: #f9fafb

## 주의사항

1. **해시 라우팅 유지**: `#my-storage` 해시 기반 SPA 네비게이션 변경 금지
2. **기존 3차 탭 구조 유지**: 내자료/담은자료/구독채널/채널관리 탭 순서 및 기능 변경 금지
3. **더미 데이터 일관성**: 통계 수치가 논리적으로 일관성 있어야 함 (예: 폴로워 증가 시 조회수도 증가)
4. **Chart.js 버전**: 기존 차트 설정 패턴 유지
5. **성능**: 콘텐츠가 1000개 이상일 경우 페이지네이션 또는 무한스크롤 추가 검토

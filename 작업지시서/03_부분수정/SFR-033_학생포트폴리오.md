## 기존 구현 현황

### 현재 보유 기능
- **통계카드 5개**: 포트폴리오 수, 평균점수, 총시간, 대표활동, 최근 업데이트
- **필터+검색**: 학년 / 학기 / 과목 드롭다운 + 텍스트 검색
- **3종 보기**: 카드 보기(그리드) / 목록 보기(테이블) / 타임라인 보기(연표)
- **포트폴리오카드**: 썸네일 이미지 + 제목 + 점수배지 + 날짜
- **인생과제 별마크**: 즐겨찾기 별 아이콘 표시
- **상세모달**: 포트폴리오 상세내용, 태그 선택 (기존)
- **사이드바**: 레이더차트(학습목표), 빠른액션 버튼

### 디자인 톤
- 색상: 파란색 주(#2563eb, #3b82f6), 초록 포인트(#10b981)
- 카드: 흰색 배경, border-radius 12px, 미세 박스섀도우
- 폰트: Pretendard, 제목 16px/600, 본문 14px/400
- 배경: #f5f7fa (페이지), #ffffff (카드)

## 추가/보완 항목

### 1. 회고/역량 태깅 강화 (필수)
**기존**: 상세모달에 단순 태그 선택 (태그 리스트 미지정)
**추가 필요**:
- **2022 개정 핵심역량 6개축 통합**:
  - 자기관리역량 (주황, #f59e0b)
  - 창의적사고역량 (파랑, #3b82f6)
  - 의사소통역량 (초록, #10b981)
  - 공동체역량 (빨강, #ef4444)
  - 정보활용역량 (보라, #8b5cf6)
  - 문제해결역량 (핑크, #ec4899)
- **포트폴리오 상세모달**에서:
  - 각 역량별 탭 또는 토글 UI 추가
  - 탭 선택 시 해당 역량 소속 "회고질문" 표시 (드롭다운)
  - 예: [자기관리역량] → "이 활동에서 어떻게 자신을 관리했나요?" (입력필드)
  - 입력 후 자동 "역량태그" 추가 (칩 형식)
- **포트폴리오 카드에 역량 아이콘 표시**: 우측상단 3~4개 역량 배지 (20x20px, 컬러)

### 2. 외부 공유/내보내기 (권장)
**추가 필요**:
- **상세모달 우측상단에 "공유" 버튼** 추가:
  - 버튼 클릭 시 선택지:
    - "링크 복사" → 공유가능 고유 URL 생성 및 클립보드 복사
    - "PDF 내보내기" → 포트폴리오 상세 PDF 다운로드 (로고/학생명/역량태그 포함)
    - "카카오톡 공유" → 카카오톡 앱으로 공유 (선택사항)
- **링크 공유 모달**: 생성된 URL + "복사" 버튼 + "만료 설정" 드롭다운 (7일/30일/무제한)
- **PDF 내보내기**: 포함항목 선택 체크박스
  - 포트폴리오 이미지 (선택)
  - 회고 및 역량 분석 (기본 선택)
  - 평가/점수 (선택)

### 3. 프라이버시 설정 (권장)
**추가 필요**:
- **포트폴리오별 공개설정** (포트폴리오 카드에 아이콘 표시):
  - 🔒 비공개 (학생+교사만 열람)
  - 👥 공개 (반 전체 열람)
  - 🌐 웹공개 (링크 공유 시 누구나)
- **상세모달에서 설정 변경** (드롭다운):
  - 설정 변경 시 확인 팝업: "비공개로 변경하면 공유된 링크 무효화됩니다"
- **민감정보 마스킹** (선택사항):
  - "점수 숨기기" 토글 (공개/공유 시 점수 표시 안됨)
  - "학생명 숨기기" 토글 (외부 공유 시 익명으로 표시)

### 4. 알림/리마인더 (선택)
**추가 필요**:
- **리마인더 설정** 버튼 (상세모달 우측):
  - "회고 작성 리마인더" → 일정 시간 후 알림
  - "피드백 받기 리마인더" → 교사에게 피드백 요청 알림
- **알림 센터 연계**: 우측상단 "알림" 아이콘에 새 알림 배지 표시
- **마지막 피드백 날짜** 표시: 상세모달에 "선생님 피드백: 3주전" 텍스트

## 디자인 참고

### 색상 정의
```
6개 핵심역량 색상:
- 자기관리역량: #f59e0b (주황)
- 창의적사고역량: #3b82f6 (파랑)
- 의사소통역량: #10b981 (초록)
- 공동체역량: #ef4444 (빨강)
- 정보활용역량: #8b5cf6 (보라)
- 문제해결역량: #ec4899 (핑크)

공개설정 색상:
- 비공개: #6b7280 (회색)
- 공개: #3b82f6 (파랑)
- 웹공개: #10b981 (초록)
```

### 컴포넌트 명세
- **역량 탭**: 높이 40px, 배경 #f3f4f6, 활성탭 아래 색상 언더라인 (3px)
- **역량 배지 (카드)**: 20x20px, 원형, 배경색 + 센터 흰색 아이콘
- **공유 버튼**: 높이 36px, 폭 80px, 테두리 스타일, 파란색 (#3b82f6)
- **공개설정 아이콘**: 카드 우측상단, 16x16px, 반투명 배경 동그라미 + 아이콘
- **역량 칩**: 높이 24px, padding 6px 12px, 배경 해당 역량색(연하게), 폰트 12px/500

### 레이아웃 조정
- 상세모달: 폭 500px → 550px (역량탭 공간)
- 포트폴리오 카드: 역량 아이콘 공간 추가 (우측상단 60x20px)

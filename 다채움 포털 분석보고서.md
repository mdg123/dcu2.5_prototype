## 1. 프로젝트 개요
### 1.1 다채움 포털이란
다채움(dcu.cbe.go.kr)은 **충청북도교육청**이 운영하는 통합 교육 디지털 플랫폼이다.
"충북의 학생이라면, 다채움으로 함께 배우고, 나답게 성장해요!"를 슬로건으로 하며,
채움클래스 / 채움콘텐츠 / 스스로채움 / 나의 성장기록 / 채움성장 / 채움더하기 등
다양한 교육 서비스를 단일 포털에서 제공한다.
### 1.2 대상 사용자
| 역할 | 설명 |
|------|------|
| 교사(교직원) | 클래스 개설, 수업·과제·평가 등록, 학생 관리, 성장기록 조회 |
| 학생 | 클래스 참여, 수업 이수, 과제 제출, 평가 응시, AI 맞춤학습 |
| 학부모 | 클래스 참여(읽기 권한), 자녀 학습 현황 조회 |
| 관리자 | 담임 교사 설정 승인, 시스템 전체 관리 |
### 1.3 전체 GNB(글로벌 내비게이션) 구조
```
다채움 포털 (dcu.cbe.go.kr)
├── 채움클래스
├── 채움콘텐츠
├── 스스로 채움
├── 우리반 성장기록
├── 채움성장 (체인지 / 기초학력 / 방과후돌봄 / 진로진학)
├── 채움더하기 (채움모니터 / 채움책방 / 채움수학 / 채움수능 / 채움코딩 / 채움타자 / 채움CBT / 채움영어 / 채움캔버스)
└── 이용안내 (공지사항 / 자주 묻는 질문 / 도움 자료실)
```
### 1.4 공통 UI 패턴
- **헤더**: 다채움 로고(좌) + 알림 / 바로가기 / 사용자 프로필(우)
- **색상 체계**: 메인 블루(#0057A8 계열) + 포인트 핑크/레드(버튼), 그린(완료 상태), 그레이(비활성)
- **레이아웃**: 상단 GNB + 컨텐츠 영역(최대 1080px 센터 정렬)
- **반응형**: PC 중심, 모바일 대응
- **공통 컴포넌트**: 카드형 목록, 원형 진행률 차트, 캘린더, 탭 내비게이션, 모달 팝업

### 2.1 채움클래스
#### 목적 및 핵심 기능 요약
교사가 온라인 클래스를 개설하고, 학생을 초대하여 수업·과제·평가·소통을 진행하는
**LMS(학습관리시스템)** 핵심 서비스이다. Google Classroom과 유사한 구조를 가진다.
#### 메뉴/페이지 트리 구조
```
채움클래스 (class.cbe.go.kr)
├── 나의 클래스 [메인]
│   ├── 캘린더 (월별 수업/과제/평가 일정)
│   ├── 바로가기 (내 클래스 목록)
│   └── 클래스 현황 (TODAY 감정출석, 수업이수율, 과제제출율, 평가진행률)
├── 클래스 찾기
│   ├── 가입가능한 클래스 탭
│   ├── 우리학교 클래스 탭
│   ├── 클래스코드로 찾기
│   └── 명예의 전당 / 가입 신청·승인 관리
├── 클래스 관리
│   ├── 현재 클래스 목록 (클래스명, 멤버수, 개설자, 상태, 게시판 유형)
│   ├── 멤버관리 / QR코드 보기
│   └── + 새로 만들기
├── 클래스별 학습분석
│   ├── 월별/주별 보기
│   ├── 수업이수율 차트
│   ├── 과제제출률 차트
│   ├── 평가풀이율 차트
│   ├── 수업 이해도 / 집중도
│   └── 탭: 수업 / 과제 / 평가 / 커뮤니티
└── 활동내역 보기
[클래스 홈 - 개별 클래스 진입 후]
├── 홈 (최근 활동분석, 오늘의 일정, 클래스 활동 분석)
├── 수업 (수업 목록, 수업 만들기, 수업 이수 현황)
├── 과제 (과제 목록, 과제 만들기, 제출 현황)
├── 평가 (평가 목록, 평가 만들기, 응시 결과)
├── 알림장 (공지 게시판)
├── 감정출석부 (학생 감정 체크인)
├── 설문 (설문 목록·생성)
├── 방명록 (자유 게시판)
└── 우리반 앨범 ♬ (미디어 게시판)
```
#### 주요 화면별 UI 구성 요소
**나의 클래스 (메인)**
- 월별 캘린더: 날짜별로 `과제` `평가` `수업` 태그 표시
- 우측 사이드바: 내 클래스 바로가기 목록 (전체/개설/참여/대기 필터)
- 하단 클래스 현황 카드 4종: TODAY 감정출석부 / 수업이수율 / 과제제출율 / 평가진행률
- 하단 피드: 새로 올라온 글 / 인기글 / 설문
**클래스 관리**
- 테이블 형태: 클래스명, 개설자, 상태(개설/참여), 게시판 유형 체크박스 표시
- 게시판 유형: 수업 / 과제 / 평가 / 알림장 / 감정출석부 / 설문 / 자유게시판 (활성 여부 표시)
- 관리 버튼: 멤버관리 / QR코드 보기 / 더보기(⋮)
**클래스 홈 (개별 클래스)**
- 상단 배너: 클래스명 + 클래스 드롭다운 전환
- 탭 내비게이션: 홈 / 수업 / 과제 / 평가 / 알림장 / 감정출석부 / 설문 / 방명록 / 우리반앨범
  - 클래스 유형에 따라 탭이 다르게 구성됨 (학년반 클래스는 전체 탭, 주제형은 선택적 탭)
- 홈 카드: 최근 활동 분석(학습시간, 출제문제수, 평균정답률) / 오늘의 일정 / 클래스 활동 분석
**클래스별 학습분석**
- 필터: 월별/주별 선택 + 클래스 선택 드롭다운
- 클래스 활동현황: 수업이수율 / 과제제출률 / 평가풀이율 (원형 게이지)
- 상세 탭: 수업 이수율 꺾은선 차트 / 수업 이해도 도넛 차트 / 수업 집중도 도넛 차트
#### 데이터 흐름
```
[교사 클래스 생성]
  → 클래스 정보 등록 (이름, 유형, 설명, 커버이미지)
  → QR코드/클래스코드 생성
  → 학생 초대 or 학생 직접 가입 신청
[수업 등록]
  교사: 수업 만들기 → 콘텐츠 첨부(채움콘텐츠에서 불러오기 or 직접 업로드) → 배포
  → 학생 [오늘의 학습]에 수업 항목 표시
  → 학생 수업 이수 → 이해도/집중도 피드백 입력
  → 교사 [클래스별 학습분석]에 이수율 반영
[과제 등록]
  교사: 과제 만들기 → 마감일 설정 → 배포
  → 학생 과제 제출 → 교사 채점/피드백
  → [우리반 성장기록] 과제제출률에 반영
[평가 등록]
  교사: 평가 만들기 (채움CBT 문항 불러오기 or 직접 생성) → 배포
  → 학생 응시 → 자동 채점
  → 오답은 [오답노트]에 자동 저장
  → [우리반 성장기록] 평가풀이율에 반영
```
#### 사용자 역할별 화면 차이
| 화면 요소 | 교사 | 학생 | 학부모 |
|-----------|------|------|--------|
| 클래스 개설 | O | X | X |
| 수업/과제/평가 만들기 | O | X | X |

| 클래스 활동 분석 | O (반 전체) | O (본인) | O (자녀) |
| 감정출석부 체크 | X (확인만) | O | X |
| 알림장 작성 | O | X | X |

### 2.2 채움콘텐츠
#### 목적 및 핵심 기능 요약
교사와 학생이 수업에 사용할 다양한 교육 자료(영상, 이미지, 문서, 문항, 평가지, 활동지, 수업꾸러미, 수업레시피)를 **생성·공유·검색·구독**하는 콘텐츠 플랫폼이다.
#### 메뉴/페이지 트리 구조
```
채움콘텐츠 (class.cbe.go.kr/mypage/myArchive.do)
├── 나의 보관함
│   ├── 내자료 탭
│   │   └── 콘텐츠 유형별: 영상 / 이미지 / 문서 / 문항 / 평가지 / 활동지 / 수업꾸러미 / 수업레시피
│   └── 담은자료 탭
│       └── 타인 콘텐츠 중 내가 담은 자료 목록
├── 추천콘텐츠
│   ├── 인기 콘텐츠 20 (최근 7일 기준 좋아요·담기 많은 순)
│   └── 콘텐츠 유형 탭 필터 (영상/이미지/문서/문항/평가지/활동지/수업꾸러미/수업레시피)
├── 공개콘텐츠
│   ├── 검색 (저작자명, 콘텐츠 키워드)
│   ├── 조건검색 (상세 필터)
│   ├── 태그/채널 필터 (다채움 예술은, 인성교육, AI 에듀테크 등)
│   └── 콘텐츠 유형 탭 필터
└── 성취기준별 콘텐츠 찾기
    └── 교과/학년/성취기준으로 콘텐츠 탐색
[콘텐츠 뷰어 / 리더]
└── /reader/readerItem.do?readerId={id}
    ├── 콘텐츠 재생/열람
    ├── 좋아요 / 담기 / 공유 / 댓글
    └── 연관 콘텐츠 추천
[새로 만들기]
└── 콘텐츠 유형 선택 → 에디터 → 공개설정 → 등록
```
#### 주요 화면별 UI 구성 요소
**나의 보관함**
- 프로필 카드: 아바타 + 이름 + 나의 구독 현황 / 내 채널 구독자 수
- 콘텐츠 통계 테이블: 유형별 등록수 / 담은수 (영상, 이미지, 문서, 문항, 평가지, 활동지, 수업꾸러미, 수업레시피)
- 내가 즐겨찾는 크리에이터 목록 + 구독하기 버튼
- 나의 콘텐츠 활동 꺾은선 차트 (조회수/공유수/담은수/댓글)
- 인기 채널 Best 섹션
**공개콘텐츠**
- 상단 통합 검색바 + 조건검색 버튼
- 콘텐츠 유형 탭: 영상 / 이미지 / 문서 / 문항 / 평가지 / 활동지 / 수업꾸러미 / 수업레시피
- 태그 클라우드 (채널/출처별 키워드 태그)
- 전체선택 체크박스 + 선택 담기 기능
- 콘텐츠 카드: 썸네일 + 제목 + 저작자 + 조회수 + 등록일 + 출처
- 하단 액션: 좋아요(♡) / 담기 / 공유 / 댓글
**추천콘텐츠**
- 인기 콘텐츠 20 슬라이더 (최근 7일 기준)
- 맞춤설정 버튼 (개인화 필터)
- 격자형 / 중간형 / 목록형 뷰 전환
#### 데이터 흐름
```
[콘텐츠 생성]
  교사/학생: 새로 만들기 → 유형 선택 → 에디터 작성 → 공개 범위 설정
  (공개 / 충북교원 / 충북학생 / 학교교원 / 학교학생 / 비공개)
  → 등록 완료 → 나의 보관함 > 내자료에 저장
[콘텐츠 활용]
  교사: 공개콘텐츠 탐색 → 담기 → 채움클래스 수업 등록 시 첨부
  → 학생에게 수업으로 배포 → 학생 열람/이수
[콘텐츠 구독]
  사용자: 크리에이터 채널 구독 → 추천콘텐츠에 해당 채널 콘텐츠 우선 노출
[성취기준 연계]
  콘텐츠 등록 시 성취기준 태그 매핑
  → [성취기준별 콘텐츠 찾기]에서 교과/학년/성취기준으로 검색 가능
```
#### 사용자 역할별 화면 차이
| 기능 | 교사 | 학생 |
|------|------|------|
| 콘텐츠 생성(전 유형) | O | O (일부 유형 제한 가능) |
| 추천콘텐츠 접근 | O | O |
| 공개콘텐츠 공개 범위 설정 | 충북교원/충북학생/학교교원/학교학생 전체 | 충북학생/학교학생 등 |
| 수업꾸러미/수업레시피 생성 | 주 대상 | 가능하나 보조적 |

### 2.3 스스로채움
#### 목적 및 핵심 기능 요약
학생이 AI 기반 자기주도 학습을 수행하는 서비스이다. 채움클래스에서 교사가 배포한 수업/과제/평가 외에도, AI가 학생 수준을 진단하여 개인 맞춤형 학습 콘텐츠를 추천하고, 오답노트로 약점을 관리한다.
#### 메뉴/페이지 트리 구조
```
스스로 채움 (class.cbe.go.kr/learn/learnMain.do)
├── 오늘의 학습 [메인]
│   ├── 주간 캘린더 (월~일 날짜 선택)
│   ├── AI 맞춤학습(수학) 카드
│   │   ├── 스스로 진단
│   │   └── 맞춤형 AI학습
│   ├── 오답노트 카드 (점검 필요한 오답노트 수)
│   └── 우측 사이드바
│       ├── 사용자 정보 (학교명)
│       ├── 이번주 진행현황 (완료한 오늘의학습 / 학습한 AI맞춤학습)
│       ├── 채움포인트

### 2.4 우리반 성장기록
#### 목적 및 핵심 기능 요약
**담임교사 전용** 서비스로, 담당 학급 학생들의 학습 활동 데이터를 종합 분석하여 제공한다. 수업이수율, 과제제출률, 평가풀이율, 감정 데이터, 포인트 적립 현황, 오답노트 등을 학생 개인별·교과목별로 조회할 수 있다. 관리자(교감/교장)의 승인 후 담임교사 권한이 부여된다.
#### 메뉴/페이지 트리 구조
```
우리반 성장기록 (class.cbe.go.kr/analytics/)
├── 우리반 학습분석
│   ├── 학생별 보기 (전체 학생 보기 버튼)
│   ├── 월별/주별 기간 필터
│   ├── 나의 요약 탭
│   │   ├── 나의 감정 (감정출석부 데이터)
│   │   ├── 포인트 적립 현황 (기간 적립 포인트 / 총 포인트)
│   │   ├── 클래스 활동현황 (수업이수율, 과제제출율, 평가풀이율)
│   │   └── 관심키워드 (학생 검색 키워드 분석)
│   └── 교과목별 분석 탭
│       └── 교과목별 학습 데이터 상세
├── 학생 오답노트
│   ├── 반 학생 전체 오답노트 목록
│   ├── 학생별 필터
│   └── 평가지별 오답 현황
└── 학생 포인트
    ├── 포인트 지급·회수 탭
    │   ├── 학생 선택 (전체 or 개별)
    │   ├── 포인트 적립/차감 입력
    │   └── 메시지 선택 (기본 메시지 4종 + 직접입력)
    └── 지급·회수 이력 탭
        └── 포인트 지급/회수 이력 조회
```

### 2.5 채움CBT (채움더하기 > 채움CBT)
#### 목적 및 핵심 기능 요약
교사가 CBT(Computer Based Test) 방식의 시험지를 제작·공유하고, 학생이 온라인으로 시험에 응시하는 **디지털 평가 플랫폼**이다. hwp/pdf 파일 업로드로 시험지를 등록하고, 학생은 코드 입력 없이 배포된 시험에 응시한다.
#### 메뉴/페이지 트리 구조
```
채움CBT (class.cbe.go.kr/publicQuiz/)
├── 채움CBT시험지 [메인]
│   ├── 검색 필터
│   │   ├── 학교급 (전체 / 초등학교 / 중학교 / 고등학교)
│   │   ├── 교과 (전체 / 국어 / 수학 / 영어 / 사회 / 과학 / 체육 / 예술 / 기술·가정 / 정보 / 기타)
│   │   ├── 검색영역 (전체 / 충청북도 전체 / 우리학교만 보기)
│   │   └── 키워드 검색 (시험지명 또는 키워드)
│   ├── 정렬 (최신 등록일 순 / 응시기간 임박순)
│   └── 시험지 카드 목록 (학교급 태그, 교과 태그, 응시기간, 시험지명, 제작교사, 학교명)
├── 나의 보관함
│   ├── 필터 (전체 / 임시저장 / 비공개 / 진행중만보기)
│   ├── 시험지 카드 (내가 등록한 시험지)
│   └── 문제풀이 결과 버튼 (제작자용 결과 조회)
├── 나의 평가결과
│   ├── 시험지 선택 드롭다운
│   ├── 응시자 정보 (이름, 학교급, 과목/평가명, 응시일)
│   ├── 채점결과 (정답수/총문항수, 정답률 %)
│   ├── 풀이 시간
│   └── 문항별 정오 현황 (O/X 표시)
├── 시험지 등록하기 (교사)
│   ├── Step 1: 시험지 및 정보 등록
│   │   ├── 제목, 학교급, 교과, 설명, 키워드
│   │   ├── 공개옵션 (공개/충북교원/충북학생/학교교원/학교학생/비공개)
│   │   ├── 파일 업로드 (hwp, hwpx, pdf)
│   │   └── 세부옵션 설정 (응시기간, 제한시간 등)
│   └── Step 2: 정답 입력
│       └── 업로드된 문항별 정답 설정
└── 시험지 불러오기
    └── 기존 시험지 복사·수정
```

| 평가 연결 | 채움CBT | 채움클래스 | 채움CBT 시험지를 클래스 평가로 등록·배포 |
| 오답 자동 저장 | 채움클래스(평가) / 스스로채움(AI학습) | 오답노트 | 오답 발생 시 자동으로 오답노트에 누적 |
| 학습 데이터 집계 | 채움클래스 + 스스로채움 | 우리반 성장기록 | 수업이수율·과제제출률·평가풀이율·감정 데이터 통합 집계 |
| 포인트 반영 | 우리반 성장기록(교사 지급) | 스스로채움(학생 화면) | 교사가 지급한 포인트가 학생 채움포인트에 실시간 반영 |
| 성취기준 연계 | 채움콘텐츠 | 채움클래스 | 성취기준 태그 콘텐츠를 수업 단원에 매핑 |
### 3.3 사용자 여정 요약 (User Journey)
```
교사 여정:
  로그인 → 채움콘텐츠에서 자료 탐색·저장
  → 채움클래스 수업/과제/평가 등록
  → 채움CBT 시험지 제작 후 평가에 연결
  → 클래스별 학습분석으로 진도 모니터링
  → 우리반 성장기록으로 학생별 종합 분석
  → 포인트 지급으로 동기부여
학생 여정:
  로그인 → 스스로채움 [오늘의 학습] 확인
  → 수업 이수 → 과제 제출 → 평가 응시
  → AI 맞춤학습으로 자기주도 학습
  → 오답노트에서 취약 개념 복습
  → 채움포인트 적립 확인
```

| 텍스트 Secondary | `#6B7280` | 부제목, 설명 |
#### 레이아웃 패턴
- **헤더**: 높이 60px, 좌측 로고 + 우측 아이콘(알림/바로가기/프로필)
- **서브헤더(GNB)**: 높이 48px, 수평 탭 메뉴 (활성 탭 하단 컬러 언더라인)
- **컨텐츠 영역**: 최대 1080~1200px, 좌우 패딩 24px
- **카드**: border-radius 12px, box-shadow subtle, 패딩 24px
- **사이드바**: 우측 고정 240~280px (나의 클래스 메인, 스스로채움 등)
- **통계 카드**: 아이콘 + 수치(대형) + 설명 + 전주 대비 변화율
#### 네비게이션 패턴
- **GNB 드롭다운**: 마우스 오버 시 하위 메뉴 펼침
- **클래스 내 탭**: 수평 탭 (홈/수업/과제/평가/알림장/감정출석부/설문 등)
- **내부 탭 전환**: 탭 활성 상태는 배경색 + 볼드 텍스트
- **브레드크럼**: 미사용 (서비스 특성상 단일 깊이 구조)
- **클래스 전환**: 상단 드롭다운 콤보박스로 클래스 간 이동
#### 공통 UI 컴포넌트
```
- 원형 게이지 (수업이수율, 과제제출률, 평가풀이율)
- 꺾은선 차트 (주간/월간 활동 추이)
- 도넛 차트 (이해도, 집중도, 감정 분포)
